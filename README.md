## Getting started

Autoinstall Hexo & NexT & NexT Website Source on Localhost:

```bash
git clone https://github.com/next-theme/theme-next-docs
cd theme-next-docs
npm install
```

Generate:

```bash
npx hexo generate
```

Run server:

```bash
npx hexo server
```

### Checkout docs for older versions

```bash
git checkout v8.0.0
npm install
npx hexo clean
```
