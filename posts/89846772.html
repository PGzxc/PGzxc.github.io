二 跨平台通用打包工具

工具 说明 支持平台
electron-builder 用于 Electron 应用的打包与发布，支持自动签名、代码签名等 Windows / macOS / Linux
Tauri CLI Rust + Webview 方案的打包工具，轻量安全 Windows / macOS / Linux
Flutter CLI flutter build 命令支持打包 Android、iOS、Web、桌面 Android / iOS / Web / macOS / Windows / Linux
React Native CLI 原生方式构建 React Native 应用，搭配 Gradle/Xcode 使用 Android / iOS
React Native CLI / EAS CLI Expo EAS CLI
Capacitor CLI Ionic 团队出品，支持 Web + 原生打包 Android / iOS / Web
Expo CLI React Native 的打包与发布增强工具 Android / iOS / Web(PWA)
Cordova CLI 老牌的 Web + 原生混合打包工具 Android / iOS
Kotlin Multiplatform Kotlin Multiplatform(Compose Multiplatform 等) Android / iOS / JVM / Web / Desktop
PyInstaller / Nuitka / Briefcase Python 应用打包为桌面应用 Win/macOS/Linux

三 Web & 桌面打包工具

工具 说明 支持平台
Vite CLI 快速构建 Web 应用，可配合 Electron/Tauri Web / Electron / Tauri
webpack CLI 配置灵活的打包工具，可定制输出格式 Web
pkg 将 Node.js 项目打包成可执行文件 Windows / Linux / macOS
nexe 类似 pkg，用于 Node.js 应用打包 Windows / Linux / macOS

四 移动端打包工具

工具 说明 支持平台

Xcode CLI (xcodebuild) iOS/macOS 项目打包与构建 iOS / macOS
Fastlane 自动化打包、签名、上传 App Store/Play 的工具 Android / iOS
HBuildX CLI 用于 uni-app 项目的打包 Android / iOS / 小程序 / Web

五 小程序/低代码工具 CLI

工具 说明 支持平台

Taro CLI 京东出品的小程序多端解决方案 微信 / RN / H5 / 支付宝等
微信开发者工具 CLI 微信官方提供的命令行工具，可用于自动预览、上传 微信小程序
