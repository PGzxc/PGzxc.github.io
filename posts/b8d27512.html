前言

本文主要讲述使用分支：

  • 查看分支：git branch
  • 创建分支：git branch
  • 切换分支：git checkout
  • 创建+切换分支：git checkout -b
  • 合并某分支到当前分支：git merge
  • 删除分支：git branch -d
