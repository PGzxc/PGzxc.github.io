React Native开发之——组件Button

发表于 2018-03-07 更新于 2020-10-29 分类于 开发 ， 移动开发 ， React Native

前言

Button-按钮，平时开发中再熟悉不过的东西了，React Native 0.37之前并没有Button按钮，一直用TouchableOpacity 或 TouchableNativeFeedback替代，React Native 0.37开始加入Button组件，今天我们要学的就是Button组件。

  • title
  • color
  • disabled
  • accessibilityLabel
  • onPress

Button

title

Title属性

设置Button上显示的文字，类型为string

color

color属性

Button没有background属性，通过color来显示Button的背景色

disabled

disabled属性

boolean类型值，设置Button是否可以点击。

accessibilityLabel

accessibilityLabel属性

辅助功能，用于显示盲人可访问性特征的文本

onPress

onPress属性

设置Button的点击事件。

实战

代码

export default class App extends Component<Props> {
render() {
    return (
        <View style={styles.container}>
            <Button
                onPress={this.onPress}
                title="Learn More"
                color="#f00"
                disabled={false}
                testID={'button'}
                accessibilityLabel="Learn more about this purple button"/>
        </View>
    );
}

onPress() {
    alert('onPress')
}
}
