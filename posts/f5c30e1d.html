一 概述

元素的内边距在边框和内容区之间。控制该区域最简单的属性是padding属性。

CSS padding属性定义元素边框与元素内容之间的空白区域。

二 CSS padding属性

CSS padding属性定义元素的内边距。padding属性接受长度值或百分比值，但不允许使用负值。
