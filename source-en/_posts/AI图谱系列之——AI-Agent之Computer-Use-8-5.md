## 三 Computer Use(终极Agent形态)

### 3.1 什么是 Computer Use

```
让 AI 直接操作电脑（鼠标/键盘/UI）
```

### 3.2 主流实现？

```
1.Claude Computer Use
特点：
-屏幕理解
-操作稳定（成熟）

2.GPT-5.4 Computer Use

特点：
-原生集成
-Agent能力更强
```

### 3.3 Computer Use 本质？

```
三件事：
-屏幕理解（Vision）
-动作生成（Action）
-状态反馈（Loop）
```

### 3.4 应用场景？

```
-自动办公（Excel / PPT）
-自动测试（UI自动化）
-RPA替代
```
