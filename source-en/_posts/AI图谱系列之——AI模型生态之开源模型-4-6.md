## 三  开源模型(开发者重点)

### 3.1 开源模型有哪些？

```
Mistral / Mixtral（MoE）
Phi（微软小模型）
Falcon

```

### 3.2 热门开源体系

```
Llama 4（开源版）
Qwen 3.5（开源）
DeepSeek（部分开源）
```

### 3.3 开源模型 vs 闭源模型？

| 维度 | 开源 | 闭源 |
| :--: | :--: | :--: |
| 成本 |  低  |  高  |
| 可控 |  高  |  低  |
| 性能 |  中  |  高  |
| 部署 | 私有 |  云  |

### 3.4 什么时候用开源模型？

```
场景：
-数据敏感（金融/政务）
-离线部署
-成本敏感
```
