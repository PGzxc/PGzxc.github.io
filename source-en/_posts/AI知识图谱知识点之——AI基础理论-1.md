## 二 AI基础理论

### 2.1 机器学习

```
1.机器学习 (Machine Learning)
让机器从数据中学习规律，不硬编码规则。

2.深度学习 (Deep Learning)
用多层神经网络从数据中自动学习特征。

3.神经网络 (Neural Network)
模拟人脑神经元结构的计算模型。

4.监督学习 (Supervised Learning)
用有标签数据训练，学习输入→输出映射。

5.无监督学习 (Unsupervised Learning)
用无标签数据找结构、聚类、降维。

6.强化学习 (Reinforcement Learning)
智能体通过试错 + 奖励学习最优策略。
```

### 2.2 神经网络结构

```
1.CNN (卷积神经网络)
擅长图像：提取局部特征、空间结构。

2.RNN (循环神经网络)
擅长序列数据，但长序列易梯度消失。

3.Transformer
基于注意力机制，并行强、长上下文强，LLM 基石。

4.Attention (注意力机制)
自动聚焦输入中重要部分，加权计算。

5.Self Attention (自注意力)
序列内部词与词之间互相关注。

6.Cross Attention (交叉注意力)
两个不同序列互相关注（如翻译、文生图）。

7.MoE (混合专家模型)
每个 token 只激活部分专家层，稀疏计算，大模型标配。
```

### 2.3 模型训练

```
1.Pretraining (预训练)
在海量数据上从头训练，学习通用知识。

2.Fine-tuning (微调)
用下游任务数据，小幅度调整预训练模型。

3.Instruction Tuning (指令微调)
用自然语言指令数据集微调，让模型听懂指令。

4.SFT (监督微调)
用标注好的「输入→理想输出」有监督微调。

5.RLHF (人类反馈强化学习)
用人类偏好数据训练奖励模型，再强化对齐。

6.RLAIF (AI 反馈强化学习)
用强 AI 代替人标偏好，成本更低、规模更大。

7.DPO (直接偏好优化)
2026 主流，直接用偏好对优化，比 RLHF 更稳更简单。

8.PEFT (参数高效微调)
只微调少量参数，节省算力。

9.LoRA (低秩适配)
最常用 PEFT，在层旁加小低秩矩阵。

10.QLoRA (量化 LoRA)
量化 + LoRA，极低显存也能微调。
```

### 2.4 模型优化

```
1.Knowledge Distillation (知识蒸馏)
大模型（老师）教小模型（学生），保留能力。

2.Pruning (模型剪枝)
移除冗余权重 / 神经元，减小模型。

3.Quantization (模型量化)
把高精度（FP32/16）转低精度（INT8/4），更小更快。

4.Model Compression (模型压缩)
剪枝 + 量化 + 蒸馏等统称，缩小模型。

5.Sparsity / MoE 稀疏激活
只激活部分参数 / 专家，推理更快、成本更低，DeepSeek、Mixtral、Qwen 核心技术。
```
