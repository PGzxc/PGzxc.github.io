## 二 概念

```
ArkClaw 是火山引擎(字节跳动旗下云与AI服务平台)推出的云端AI智能体（Agent）服务
本质上是 OpenClaw 的云SaaS托管版本(俗称“小龙虾”的云端版)。

它参考并优化了开源的 OpenClaw 和类似产品 QClaw(腾讯系一键部署工具)，
主打零门槛、轻量化、低成本
专为个人用户和轻办公场景设计，帮助用户自动化处理日常事务和简单办公任务
```

## 三 核心定位与特色

### 3.1 核心定位与特色

```
1.开箱即用，无需部署：
无需本地安装服务器、编写代码或手动配置API Key。
一键创建后即可7×24小时在线，通过网页直接对话下达自然语言指令，ArkClaw就会自动执行任务。

2.集成火山方舟（Doubao）大模型：
无缝使用订阅的火山方舟 Coding Plan，告别按Token量计费的焦虑，
采用固定套餐计费模式，更适合长期使用。

3.多渠道交互：
支持网页端操作，还深度集成飞书（Lark）OpenClaw官方插件，
并兼容多种主流即时通讯APP，让使用更丝滑
(例如在飞书里直接指挥任务，无需反复切换上下文)。

4.专属资源：
提供一对一专属ECS云资源，安全可信，支持多智能伙伴协同工作。

5.适用场景：
 -轻量办公：自动生成周报、整理文档、同步表格、预约会议等。
 -个人事务：定时提醒、文件整理、票务预订、节日祝福、日程管理、内容优化、学习辅助等。
```

### 3.2 与OpenClaw/QClaw的区别

```
OpenClaw是开源自托管版，需要本地环境；
QClaw更侧重一键本地部署；
而ArkClaw是纯云端SaaS，门槛最低，适合不想折腾硬件的普通用户。

相比本地方案，它更稳定、无需维护服务器。
```

### 3.3 软件端

```
目前没有独立的桌面软件端(Desktop客户端)，主要通过网页控制台和飞书插件使用，操作轻便。
```

## 四 定价与订阅(付费模式)

### 4.1 付费说明

```
ArkClaw 采用预付费套餐化计费，与火山方舟 Coding Plan 深度绑定（非独立按量计费）。
具体价格以官方控制台为准，存在首购特惠和新购优惠：
```

### 4.2 付费套餐

```
1.Lite 套餐（入门级）：
约 9.9元 首购（常规约40元/月），附赠7天 ArkClaw 体验（部分时期新购Lite不再赠送，需注意）。
每5小时约1200次请求限制，适合轻度用户试用。

2.Pro 套餐（推荐）：
约 49元/月 首购（常规更高，如200元/月级别）
订阅期间可畅享 ArkClaw 服务，无额外Token焦虑，适合日常重度使用。
```

### 4.3 注意事项

```
体验期或订阅到期后，ArkClaw 实例会关停/删除，数据保留24小时。
支持退订，但有规则限制。
首购优惠和具体规格请登录火山引擎控制台查看最新活动。
整体价格在同类“大厂Claw”产品中属于亲民水平(对比本地Mac Mini或人工代装方案便宜很多)

官网地址(需登录火山引擎账号)：https://console.volcengine.com/arkclaw/welcome
```

## 五 简单教程

```
1.订阅前置：
先登录火山引擎账号，进入“火山方舟”订阅 Coding Plan（Lite/Pro）。

2.创建 ArkClaw：
在控制台左侧导航选择 “Agent > ArkClaw”，点击“立即开始”或“立即创建”，
按照提示配置套餐(组合购买更优惠)。

3.使用：
创建成功后，通过网页对话框输入自然语言指令（如“帮我整理本周工作总结”或“提醒我明天买票”）。
支持SkillHub技能市场一键安装扩展技能。

4.飞书集成(推荐)：
安装飞书 OpenClaw 官方插件，即可在飞书内直接与ArkClaw交互，更适合办公场景。
```
