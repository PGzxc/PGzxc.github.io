## 二 Bootstrap概念

* Bootstrap是一个使用HTML、CSS和JavaScript框架的前端开发框架
* Bootstrap 是全球最受欢迎的前端框架，用于构建响应式、移动设备优先的网站
* 简洁、直观、强悍的前端开发框架，让web开发更迅速、简单

## 五 Bootstrap各版本有什么不同

### 5.1 Bootstrap 2

* 针对桌面端设计；
* 包含基本的 CSS、JS 组件，比较简单；
* 没有响应式布局；
* 需要兼容 IE6

### 5.2 Bootstrap 3

* 添加了响应式布局，支持移动设备；
* 扁平化设计，比较简洁明了；
* 可以配合 SASS、LESS 等预处理器使用，增加了定制化选项；
* 支持 IE8+

### 5.3 Bootstrap 4

* 改进了组件和样式，添加了新的组件；
* 使用了更少的代码，加载速度更快、易于定制；
* 可以自定义网格系统；
* 支持 IE9+

### 5.4 Bootstrap 5

* 支持更多的现代浏览器，而 IE 系列并不再支持；
* 移除了 jQuery 依赖；
* 优化了网格系统和排版；
* 更轻量级，代码更加简洁
