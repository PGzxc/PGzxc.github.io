## 二 前置准备

```
-请先注册在线渠道
-并添加api-key
```

## 三 在 CC Switch 中添加步骤

### 3.1 切换应用

```
打开 CC Switch，切换到你要使用的应用（如 Claude Code）。
```

### 3.2 点击右上角

```
点击右上角 +（Add Provider / 添加供应商）。
```

![][1]

### 3.3 供应商选择DeepSek

```
预设供应商：DeekSeek
```

![][2]

### 3.4 填写以下关键信息

```
填入API Key
```

图示

| 1-API key | 2-默认配置 |
| :-------: | :--------: |
|  ![][3]   |   ![][4]   |

### 3.5 保存

```
点击 添加 或 保存
```

### 3.6 选中配置

```
在供应商列表中选中这个新配置，点击 启用（或「使用」）。
```

### 3.7 测试

```
打开终端，输入 claude（或对应 CLI 命令），然后试用模型。
由于没有额度了，返回异常
```
