## 二 需求说明

```
1.很多时候我们会遇到这种需求：
-从 视频A 中提取音频
-将提取出来的音频
-合成到 视频B 中

2.例如：
-替换视频背景音乐
-修复视频无声问题
-将录音配到视频画面
-将影视片源 + 原声音轨重新组合

本文介绍 最简单、稳定的三种方法。
```

## 四 方法1：使用 FFmpeg(最推荐）)

优点：不重新编码、速度快、画质无损

### 4.1 安装 FFmpeg

```
1.下载后解压：例如 C:\ffmpeg\
2.把 bin 目录加入系统环境变量。
3.测试：ffmpeg -version 出现版本号说明安装成功。
```

### 4.2 提取视频中的音频

```
假设：视频A：videoA.mp4

1.执行：

ffmpeg -i videoA.mp4 -vn -acodec copy audio.aac

2.解释：

-i videoA.mp4   输入视频
-vn             不要视频
-acodec copy    直接复制音频
audio.aac       输出音频

3.这样就提取出音频文件。
```

### 4.3 将音频合成到另一个视频

```
假设：
视频B：videoB.mp4
音频：audio.aac

1.执行：

ffmpeg -i videoB.mp4 -i audio.aac -c:v copy -map 0:v:0 -map 1:a:0 -shortest output.mp4

2.解释：

-c:v copy      视频不重新编码
-map           指定视频和音频轨
-shortest      以最短轨道为结束

3.输出文件：

output.mp4
```

## 五 方法2：格式工厂(图形界面)

适合不熟悉命令行的用户。

### 5.1 提取音频

```
打开软件

1.选择：音频 → MP3
2.添加视频文件：videoA.mp4
3.点击：开始
4.即可生成：audio.mp3
```

### 5.2 合成视频

```
1.打开：高级 → 视频合并

2.步骤：
-1 添加视频B
-2 添加音频
-3 设置音频轨

3.导出视频即可。
```

## 六 方法3：剪映(最简单)

适合完全新手

### 6.1 导入素材

```
打开剪映：
-导入 videoA
-导入 videoB
```

### 6.2 提取音频

```
右键：videoA → 提取音频
生成：音频轨道
```

### 6.3 合成

```
步骤：

-删除 videoA 视频轨
-保留音频
-将 videoB 拖入时间轴
-调整音频位置即可。
```

### 6.4 导出视频

```
点击：导出 完成。
```

## 八 常见问题

### 8.1 音频不同步

```
1.原因：
-视频帧率不同
-音频长度不同

2.解决：
-shortest
或在剪辑软件中手动调整。
```

### 8.2 没有声音

```
1.可能原因：音频格式不兼容
2.解决：转成 AAC 或 MP3
```

### 8.3 音频太短

```
可以循环：
ffmpeg -stream_loop -1
```
