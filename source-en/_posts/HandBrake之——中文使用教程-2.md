## 一 概述

```
适用系统：Windows / macOS / Linux
目标：让你快速学会如何用 HandBrake 转码、压缩、加字幕
```

## 二 下载与安装

### 2.1 官网地址

```
https://handbrake.fr/downloads.php
```

### 2.2 下载选择

```
进入官网后选择你的系统版本：
 -Windows → 下载 .exe 安装包
 -macOS → 下载 .dmg 安装包
 -Linux → 可通过命令行或 Flatpak 安装
```

### 2.3 安装完成后启动软件，主界面如下

```
+---------------------------------------------+
| 源文件（Source） | 预设（Preset） | 队列（Queue） |
+---------------------------------------------+
| [打开源视频]  [选择输出目录]               |
| [格式: MP4/MKV/WebM] [视频编码器:H.264]    |
| [质量滑杆: RF20] [添加到队列 / 开始编码]   |
+---------------------------------------------+
```

## 三 基础使用

### 3.1 打开源文件

```
点击「打开源」（Open Source），选择一个视频文件或文件夹（可是 DVD/蓝光结构）
```

### 3.2 设置输出路径

```
在「保存为」（Save As）栏选择输出目录。
```

### 3.3 选择格式与编码(常用推荐设置)

|    项目    |            建议值            |
| :--------: | :--------------------------: |
|    容器    |             MP4              |
| 视频编码器 | H.264 (x264) 或 H.265 (x265) |
| 音频编码器 |             AAC              |
|    字幕    |   可嵌入或外挂 .srt 文件   |
|    质量    |  RF 20（数值越小画质越高）   |

### 3.4 使用设备预设

```
右侧「Presets」区域，可快速选择目标设备：
 -Apple 720p30 / Android 1080p / Roku 2160p
 -Web Optimized（网页播放优化）
```

### 3.5 开始编码

```
确认设置后，点击上方「开始编码（Start Encode）」即可。
多个文件可使用「添加到队列（Add to Queue）」进行批量处理。
```

## 四 高级设置

|    功能    |                             说明                             |
| :--------: | :----------------------------------------------------------: |
|  视频滤镜  |     去交错（Deinterlace）、降噪（Denoise）、裁剪（Crop）     |
| 比特率控制 | 支持恒定质量（Constant Quality）或平均比特率（Average Bitrate） |
|    字幕    |              可导入 SRT、嵌入或烧录（Burn in）               |
|    音频    |            可复制源音轨或重新编码为 AAC、AC3、MP3            |
|    章节    |                可保留原始章节或自定义章节标签                |
