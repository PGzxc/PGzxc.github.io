## 二 JVSClaw是什么？

### 2.1 官网地址

```
https://jvs.wuying.aliyun.com/
```

### 2.2  JVSClaw 是什么

```
一句话理解：
JVSClaw = AI Agent + 云电脑 + 自动执行系统

它不是普通 ChatGPT，而是：
“你说一句话，它帮你真的去操作电脑”
```

### 2.3 费用

```
- 免费试用：7天
- 匠心版：约39元/月
- 大师版：约219元/月 
```

### 2.4 核心能力

1-核心能力

```
- AI智能体（Clawbot）
- 云端独立环境（ClawSpace）
- 自动执行任务（不是只回答）
```

2-举个例子

|    你说     |        它做         |
| :---------: | :-----------------: |
|  做一个PPT  | 自动打开工具 + 生成 |
| 写Excel报表 |  自动创建 + 填数据  |
|  发公众号   |   自动排版 + 发布   |

3-本质

```
本质：对话 → 操作系统 → 完成任务

官方核心定位：“用自然语言驱动 AI 在云端执行真实操作” 
```

## 三 核心架构(理解它为什么强)

### 3.1 Clawbot(你的AI助手)

```
- 每个用户可以创建一个或多个
- 有“技能 + 记忆 +执行能力”
```

### 3.2 ClawSpace(云端沙箱)

```
- 独立云电脑（隔离环境）
- AI在里面“真实操作系统”

好处：
- 不污染本地环境
- 更安全（隔离执行） 
```

### 3.3 Skills(技能系统)

```
- 内置：Word / Excel / PPT
- 扩展：ClawHub技能市场

相当于：AI + 插件生态（类似 GPTs + Tool）
```

### 3.4 执行机制(关键点)

```
不只是“生成代码”，而是：
- 控制鼠标
- 打开软件
- 执行流程
- 自动完成任务

可以理解为：RPA + AI Agent + 云桌面
```

## 四 安装与使用与快速入门

### 4.1 安装(3分钟上手)

```
方式1：网页（推荐）
打开官网直接用

方式2：客户端
- 手机：App Store / 安卓
- PC：官网下载

特点：
- 不需要复杂配置
- 不需要API Key 
```

### 4.2 快速入门(核心流程)

```
1.创建 Clawbot

进入后：创建 Clawbot → 选择云端 or 本地 → 输入名称
建议：新手选：云端

2. 选择环境
- 云端（推荐）
- 本地（进阶）

云端优势：
- 不吃你本机性能
- 自带算力
- 更安全 

3. 开始对话执行

示例：
帮我做一份产品分析PPT

AI会：
-打开PPT工具
-生成内容
-自动排版
-输出文件
```

## 六 JVSClaw vs 传统AI

### 6.1 对比

|  对比  | ChatGPT / Copilot | JVSClaw  |
| :----: | :---------------: | :------: |
|  输出  |       文本        | 实际操作 |
|  能力  |       建议        |   执行   |

JVSClaw 是“AI员工”，不是“AI助手”
