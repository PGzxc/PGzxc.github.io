## 二 什么是 Jupyter

### 2.1 官网地址

```
https://jupyter.org/
```

### 2.2 什么是 Jupyter

```
1.Jupyter
- Jupyter 是一个交互式计算平台，支持多语言(Python/R/Julia 等) 
- 核心理念：Notebook(计算笔记本)

2.Notebook 是：
- 代码 + 文本 + 可视化 + 交互 控件的组合文档 
- 可执行、可分享、可复现
```

## 三 Jupyter 生态结构

### 3.1 核心组件

```
- Jupyter Notebook
- JupyterLab
- Kernel（内核）
- Notebook 文件（.ipynb）
```

### 3.2 核心机制

```
前端 UI + 后端 Kernel 执行代码 
```
