## 二 安装方式(官方)

```
推荐方式一：pip
pip install notebook

推荐方式二：JupyterLab
pip install jupyterlab
```

## 三 启动服务

```
jupyter notebook
# 或
jupyter lab

浏览器访问：http://localhost:8888
```

## 四 核心目录说明

```
- Notebook 文件：.ipynb
- 配置目录：~/.jupyter
```
