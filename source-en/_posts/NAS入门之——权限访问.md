## 一 概述

```
本文介绍：
- NAS 设置不同账户访问不同内容，核心是权限控制 + 用户分组 + 共享文件夹授权
- 以最常见的几种 NAS 系统为例（群晖、威联通、西部数据 My Cloud、绿联、华为、飞牛等）
```

<!--more-->

## 二 核心思路(通用)

### 2.1 访问权限控制

```
NAS 的访问权限控制通常遵循三层逻辑：

1-用户（User）：每个人一个独立账号；
2-用户组（Group）：方便批量分配权限；
3-共享文件夹（Shared Folder）：每个文件夹可以单独设置访问权限。
```

### 2.2 实现目标

```
账号 A → 访问文件夹 A
账号 B → 访问文件夹 B
管理员 → 访问全部内容
```

## 三 不同 NAS 系统设置方式

### 3.1 群晖 Synology DSM

```
1、说明
DSM 系统权限设置最直观。

2、步骤

2-1、控制面板 → 用户与群组
-新建用户 A、B
-可选：建立用户组（如 “家人组”、“访客组”）

2-2、共享文件夹
-创建 “视频区”、“资料区”等不同内容文件夹

2-3、设置权限
在文件夹的权限页面中：
-用户 A：只勾选 “视频区” → 读写权限
-用户 B：只勾选 “资料区” → 只读/读写
-管理员：全部勾选

2-4、访问方式
本地访问：\\NAS_IP\文件夹名
Web 访问：https://nas_ip:5001 登录 DSM
手机 App：DS File、DS Video

3、支持高级功能：
-权限继承、加密文件夹
-不同用户使用不同共享链接
-用户配额限制（每人空间大小）
```

### 3.2  威联通 QNAP QTS / QuTS Hero

```
1、说明
类似群晖，但更偏向企业风格。

2、步骤
2-1、控制台 → 权限设置 → 用户：新建账号
2-2、共享文件夹：创建不同目录
2-3、设置访问权限：可分别设定「读取 / 写入 / 拒绝」
2-4、QNAP 特有功能
-支持基于 IP 的访问限制
-支持 Windows AD / LDAP 权限同步
```

### 3.3 西部数据 My Cloud EX2 Ultra

```
1、说明
家用为主，界面简单，权限控制略弱，但仍可实现分账户访问。

2、步骤
2-1、登录 Web 管理界面（默认 http://mycloudex2.local）
2-2、左侧菜单 → 用户 (Users) → 添加不同用户
2-3、左侧菜单 → 共享 (Shares) → 新建不同共享目录
2-4、点击每个共享目录 → 权限设置
-用户 A：访问目录 A（读写）
-用户 B：访问目录 B（只读或读写）
2-5、客户端访问：
-Windows：\\MyCloudEX2Ultra\ShareA
-Mac：smb://MyCloudEX2Ultra/ShareA

3、建议：
-若有重要资料，开启 “Private Share” 模式
-外网访问可通过 My Cloud Home 应用或 Web 端登录
```

### 3.4 绿联 / 飞牛 / 华为 NAS

```
1、说明
多为简化型系统，权限功能有限但仍能区分账户。

2、通用做法
-管理界面中新建账户；
-每个共享目录设置“访问权限”；
-不同用户登录后，仅显示授权的目录。

3、注意：
绿联支持“家庭组”管理，飞牛支持 SMB/FTP 用户隔离。
```

## 四 进阶技巧

|        目标        |                     方法                      |
| :----------------: | :-------------------------------------------: |
|  限制用户上传类型  |         通过 SMB 权限 + 文件系统过滤          |
|  某人只看媒体内容  |    将视频放入 Plex / Emby 服务，设独立账号    |
|  不同用户远程访问  |         通过 DDNS + HTTPS + 用户认证          |
| 同一文件夹不同权限 | 群晖/威联通支持「ACL 权限」，可细到子文件级别 |
|     加密文件夹     |           DSM、QTS、华为 NAS 均支持           |

## 五 简单示例

### 5.1 假设你有 3 个用户：

|  用户  |  角色  |     访问权限     |
| :----: | :----: | :--------------: |
| admin  | 管理员 |       全部       |
| father |  家人  | 家庭视频（读写） |
|  son   |  孩子  |  动漫区（只读）  |

### 5.2 群晖设置举例

```
控制面板 → 共享文件夹 → 家庭视频 → 权限
- admin：读写
- father：读写
- son：无

控制面板 → 共享文件夹 → 动漫区 → 权限
- admin：读写
- father：只读
- son：只读
```
