## 二 什么是 QClaw

### 2.1 概念

```
1.概念
QClaw 是由 腾讯 推出的 AI Agent 工具，

2.核心定位是：
“在你电脑上运行，并可通过外部入口控制的智能执行助手”

3.它的本质不是单纯聊天 AI，而是：
-能执行任务（Agent）
-能操作本地环境
-能远程下达指令
```

### 2.2 官网地址

```
https://qclaw.qq.com/
```

## 三 QClaw vs AutoClaw

|  对比项  |  AutoClaw  |    QClaw     |
| :------: | :--------: | :----------: |
| 部署方式 | 本地客户端 |  本地客户端  |
| 控制方式 | 软件内操作 | 微信远程控制 |
| 执行方式 |    本地    |     本地     |
| 远程能力 |    一般    |      强      |
| 使用入口 |    App     |     微信     |

## 四 核心能力解析

### 4.1 AI Agent 自动执行

```
QClaw具备典型 Agent 能力：

-理解任务
-拆解步骤
-自动执行
-输出结果
```

### 4.2 本地执行能力

```
1.QClaw可以：
-在本地生成文件
-执行实际任务流程
-利用电脑环境资源

2.AI不只是回答，而是“真正干活”
```

### 4.3 远程控制能力

```
1.通过微信：
-随时发送任务
-不需要打开客户端界面

2.类似：
-手机 = 控制台
-电脑 = 执行器
```
