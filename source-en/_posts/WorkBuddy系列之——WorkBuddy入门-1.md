## 二 什么是 WorkBuddy

### 2.1 概念

```
WorkBuddy 是腾讯推出的 AI 原生桌面智能体工作台，
核心定位：“坐在你电脑里的 AI 同事"
```

### 2.2 区别传统 AI

|       类型        |            能力             |
| :---------------: | :-------------------------: |
| ChatGPT / Copilot |          对话建议           |
|     WorkBuddy     | 直接帮你干活（执行 + 交付） |

## 三 核心能力(3大本质)

### 3.1 自然语言 → 自动执行

```
- 一句话描述需求
- 自动拆解任务 → 执行 → 输出结果

不需要 Prompt 工程
```

### 3.2 本地电脑操作能力

```
- 读取/修改文件
- 批量处理 Word / Excel / PDF
- 自动生成 PPT / 报表

真正“操作电脑”，不是聊天 
```

### 3.3 多 Agent 并行执行

```
- 多任务同时执行
- 自动分工

类似“AI团队”模式 
```

## 四 一句话总结

```
输入需求 → 直接交付结果（不是建议）
```
