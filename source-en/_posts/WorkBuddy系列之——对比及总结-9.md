## 三 总结

### 3.1 推荐使用方式

```
把 WorkBuddy 当成“执行型 AI”，不是聊天工具
```

### 3.2 最佳实践

```
1. 直接描述目标（不要写 prompt）
2. 给清晰输入（文件/数据）
3. 让它自己规划执行
```

### 3.3 一句话总结

```
WorkBuddy = AI执行引擎 + 自动办公操作系统
```
