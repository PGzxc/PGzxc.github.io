## 二 是什么

### 2.1 项目地址

```
https://github.com/dsphper/lanhu-mcp
```

### 2.2 是什么

```
Lanhu-MCP(蓝湖MCP服务器)是一个基于 MCP(Model Context Protocol)的 AI 协作服务，

用于：让 AI 编程工具直接读取蓝湖设计稿 + 需求文档 + 团队知识
```

### 2.3 一句话总结

```
让所有 AI 工具共享同一份需求 + 设计 + 知识库
```

## 三 核心定位

```
设计协作平台（蓝湖） + MCP协议 + AI IDE（Cursor/Claude）
= AI自动分析需求 + 自动生成代码 + 团队知识共享

官方强调：这是 AI 编程时代的“团队知识中枢”
```

## 四 解决什么问题？

### 4.1 传统开发

```
每个人的 AI 都要重复分析需求
设计稿 → 手动量尺寸、切图
无团队知识沉淀
```

### 4.2 Lanhu-MCP

```
AI 自动解析需求 + 原型
自动提取 UI 参数 + 切图
所有 AI 共享分析结果
```

### 4.3 本质

```
本质：AI 协作中台
```

## 五 核心能力

### 5.1 智能需求分析（最核心）

```
支持 Axure 原型解析：
 -开发视角（字段 / 逻辑 / 流程）
 -测试视角（测试用例 / 边界）
 -探索视角（模块结构）

准确率 > 95%
```

### 5.2 UI设计自动解析

```
自动下载设计稿
提取：尺寸 / 间距 / 颜色 / 字体
自动生成：HTML + CSS 代码
```

### 5.3 AI团队知识库(最创新)

```
AI 共享上下文
需求分析一次 → 全团队复用
留言板（类似 Jira + Slack）
```

### 5.4 多AI工具接入

```
支持：
-Cursor
-Claude Code
-Windsurf
-Cline
-通义灵码 等

所有支持 MCP 的工具都能接入
```

## 六 整体架构

```
蓝湖 (设计/需求)
      ↓
Lanhu-MCP Server（中枢）
      ↓
AI IDE（Cursor / Claude / Windsurf）
      ↓
开发 / 测试 / 产品
```
