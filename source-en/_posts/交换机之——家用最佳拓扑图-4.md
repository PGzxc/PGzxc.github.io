## 二 家庭标准拓扑(适合 90% 家庭)

### 2.1 方案 A：基础家用(入门级)

```
1、适合：普通家庭、房间不多、需求简单、没有 NAS 或只用千兆网络。

2、拓扑图
光猫（桥接）
       ↓
主路由器（WiFi6/7）
       ↓
千兆交换机（8口）
       ├── 客厅电视
       ├── 主卧 AP（面板 AP）
       ├── 次卧 AP（面板 AP）
       ├── 摄像头
       └── 电脑 / 打印机
3、优点：
-极其稳定
-全屋有线回程 + 无线 AP 覆盖
-成本低
-适合小户型

4、建议：
-有线回程（网线）必须提前布
-每个房间墙上预留双网口（电脑 + AP）
```

## 三 中高端家庭拓扑(NAS 用户最佳)

### 3.1 方案 B：进阶(NAS + WiFi 全覆盖 + 2.5G)

```
1、适合：NAS 用户、WiFi7、家庭影院、需要高速局域网传输（备份/4K/8K 文件）。

2、拓扑图

光猫（桥接）
       ↓
主路由器（1×2.5G WAN + 多 LAN）
       ↓
2.5G 交换机（8口）
       ├── NAS（2.5G/10G）
       ├── 客厅 AP
       ├── 主卧 AP
       ├── 次卧 AP
       ├── 书房电脑（2.5G）
       ├── 电视/机顶盒
       └── 摄像头（如需 PoE 用 PoE 交换机）

3、优点：
-局域网轻松达到 280MB/s（备份快几倍）
-多 AP，有线回程稳定
-NAS 高速访问
-WiFi7 体验极佳

4、标配设备：
-2.5G 交换机（6~8 口）
-WiFi6/6E/7 AP（墙上安装或吸顶）
-NAS 至少用 2.5G 网口

5、推荐布线：
-从弱电箱到每个房间走 2 根网线（电脑 + AP）
-书房 → 尽量 2 根网线（NAS / 电脑）
```

## 四 影音工作室 / 高阶 NAS / 10G 环境(发烧友)

### 4.1 方案 C：高阶(10G + 多 AP + 分层网络)

```
1、适合：摄影/视频工作室、家庭影音、重度 NAS、ProRes/RAW 视频编辑。

2、拓扑图

光猫（桥接）
       ↓
主路由器（2.5G / 10G WAN）
       ↓
核心 10G 交换机（SFP+ / RJ45）
       ├── NAS（10G）
       ├── 电脑 / 服务器（10G）
       ├── 游戏电脑（2.5G）
       ├── 2.5G 下级交换机（接入层）
       │        ├── 客厅 AP
       │        ├── 主卧 AP
       │        ├── 次卧 AP
       │        └── 摄像头
       └── 多媒体间 / 工作区 10G 回程


3、优点：
-10G 文件速度可达 800MB/s–1GB/s
-多设备并发无压力
-AP 全屋统一管理（如华为/TP-LINK/H3C/Ubiquiti）
-适合高并发、专业应用

4、布线需求：
-主机房（弱电箱或书房）需要至少 3–5 根高质量网线（CAT6A 或光纤）
-工作区预留 10G 回程
-AP 位置用 2 根网线（数据+备用/PoE）
```

## 五 分房间布线建议(最实用表格)

### 5.1 房间布线

|     房间      | 网线数量 |           用途           |
| :-----------: | :------: | :----------------------: |
|     客厅      |  2–3 根  | 电视、AP、游戏机、机顶盒 |
|     主卧      |   2 根   |        AP + 电脑         |
|     次卧      |   2 根   |        AP + 电脑         |
|     书房      |  3–4 根  |     NAS + 电脑 + AP      |
| 餐厅/客厅吊顶 |  1–2 根  |    吸顶 AP（如使用）     |
|     阳台      |   1 根   |          摄像头          |
|   入户门口    |   1 根   |     门口摄像头、对讲     |

### 5.2 最佳布线

```
CAT6A（6类增强） 最稳
用 双网线策略（防备 1 根坏了）
所有网线集中回到弱电箱或书房
```
