## 二 费用对比

### 2.1 中国电信

| 带宽  |       年费（元）       |       说明       |
| :---: | :--------------------: | :--------------: |
| 200M  |      约 940 元/年      |      单宽带      |
| 300M  | 约 480 元/年（优惠价） |  需配合融合套餐  |
| 500M  |      约 720 元/年      | 单宽带或融合优惠 |
| 1000M |     约 1200 元/年      |     千兆光纤     |

注：

```
电信常见套餐是“融合宽带”，例如手机+宽带+电视月付99元起，折算年费约1188元左右。
```

### 2.2 中国移动

|   带宽   |    年费（元）    |     说明      |
| :------: | :--------------: | :-----------: |
|   300M   |   约 360 元/年   |    单宽带     |
|   500M   |   约 480 元/年   |    单宽带     |
|  1000M   |   约 800 元/年   |    单宽带     |
| 融合套餐 | 每月约 69～99 元 | 含手机卡+宽带 |

注：

```
移动宽带普遍比电信、联通便宜，适合对稳定性要求不高的家庭使用。
```

### 2.3 中国联通

|     带宽      |  年费（元）   |  说明  |
| :-----------: | :-----------: | :----: |
|      50M      | 约 588 元/年  | 单宽带 |
|     100M      | 约 828 元/年  | 单宽带 |
|     200M      | 约 948 元/年  | 单宽带 |
|     300M      | 约 1078 元/年 | 单宽带 |
|     500M      | 约 1188 元/年 | 单宽带 |
| 1000M（千兆） | 约 1298 元/年 | 单宽带 |

注：

```
部分融合套餐（含手机+电视）价格会低于单宽带，可到营业厅咨询“智慧家庭”套餐。
```

## 三 总结建议

|  运营商  |        优点        |    缺点    |        适合人群        |
| :------: | :----------------: | :--------: | :--------------------: |
| 中国移动 |      价格便宜      | 稳定性略差 |   农村家庭、普通上网   |
| 中国联通 |      网络稳定      |  价格中等  |    普通家庭办公娱乐    |
| 中国电信 | 稳定性最强、售后好 |  价格略贵  | 对网络质量要求高的用户 |
