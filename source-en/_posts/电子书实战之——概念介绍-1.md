## 一 电子书 App 核心模块拆解

```
一个标准电子书 App，至少包含 6 个核心模块：

1.内容解析(不同格式)
2.内容渲染(字体/背景/翻页)
3.目录/章节结构
4.章节跳转 & 进度管理
5.在线加载 & 离线缓存
6.阅读设置持久化
```

<!--more-->

## 二 支持的电子书格式

### 2.1 最推荐(主流)

| 格式 |    说明    |    适合    |
| :--: | :--------: | :--------: |
| TXT  |   纯文本   |    小说    |
| EPUB | HTML + CSS | 标准电子书 |
| HTML |  直接渲染  |  在线内容  |
| PDF  |  固定排版  |  技术文档  |

### 2.2 说明

```
90% 电子书 App 主力：TXT + EPUB
```

## 三 内容显示 & 样式调整(字体 / 背景)

### 3.1 TXT 内容显示

1、基本思路

```
TXT 文件
 ↓
读取字符串
 ↓
分章节（正则 / 规则）
 ↓
分页（按屏幕尺寸）
 ↓
渲染 TextView / Canvas
```

2、字体 / 背景调整方式

|   功能   |      实现方式       |
| :------: | :-----------------: |
| 字体大小 | textSize / fontSize |
|   行距   |     lineSpacing     |
|   字体   |  自定义 TTF / OTF   |
|  背景色  |      View 背景      |
| 夜间模式 |      切换主题       |

3、 推荐做法

```
统一封装一个 ReaderStyleConfig：
data class ReaderStyle(
    val fontSize: Float,
    val lineHeight: Float,
    val backgroundColor: Int,
    val textColor: Int,
    val fontPath: String?
)
```

### 3.2  EPUB / HTML 内容显示

1、正确姿势：WebView / WKWebView

```
1、EPUB 本质
EPUB
 ├── chapter1.xhtml
 ├── chapter2.xhtml
 ├── style.css
 └── toc.ncx

2、渲染流程
解压 EPUB
 ↓
读取章节 HTML
 ↓
注入 CSS
 ↓
WebView 渲染
```

2、样式动态调整(核心技巧)

```
1、说明：
通过 JS + CSS 注入

2、示例
body {
  font-size: 18px;
  line-height: 1.8;
  background-color: #f6f1e7;
  color: #333;
}

document.body.style.fontSize = '20px'
document.body.style.backgroundColor = '#000'
document.body.style.color = '#ccc'
```

3、优势

```
支持复杂排版
支持图片、超链接
目录定位精准
```

## 四 目录结构 & 章节跳转

### 4.1 TXT 目录生成

```
1、常见正则
^(第.{1,9}[章回节]).*$

2、生成目录结构
[
  { "title": "第一章", "startOffset": 1024 },
  { "title": "第二章", "startOffset": 8096 }
]

3、跳转方式
-按 字符偏移
-或 页码索引
```

### 4.2 EPUB 目录(标准)

1、EPUB 自带目录文件

|   文件    |  作用  |
| :-------: | :----: |
|  toc.ncx  | 旧标准 |
| nav.xhtml | EPUB3  |

2、解析后结构

```
[
  { "title": "Chapter 1", "href": "ch1.xhtml#p1" },
  { "title": "Chapter 2", "href": "ch2.xhtml" }
]
```

3、跳转实现

```
location.href = 'ch2.xhtml'
或 WebView：
webView.loadUrl("file:///.../ch2.xhtml")
```

## 五 翻页 / 滚动模式

### 5.1 常见阅读模式

|   模式   |          实现           |
| :------: | :---------------------: |
| 上下滚动 |  ScrollView / WebView   |
| 左右翻页 | ViewPager / 自绘 Canvas |


## 六 在线 & 离线显示方案

### 6.1 在线阅读

1、说明

```
API 获取章节列表
 ↓
API 获取章节内容
 ↓
HTML / JSON
 ↓
渲染
```

2、优缺点

```
1、优点
-节省存储
-实时更新

2、缺点
-依赖网络
```

### 6.2 离线阅读(必须支持)

1、缓存策略(推荐)

|    内容    |   缓存方式    |
| :--------: | :-----------: |
| TXT / EPUB |   文件存储    |
|    HTML    |   本地文件    |
|    图片    |  Disk Cache   |
|  阅读进度  | SQLite / MMKV |

2、原理

```
首次在线 → 下载 → 本地缓存 → 离线可读
```

3、说明

```
EPUB 离线 = 解压后直接 file:// 加载
```
