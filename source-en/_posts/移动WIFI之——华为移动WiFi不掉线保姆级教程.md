## 二 当前联网情况

```
1.设备：华为移动WiFi 3 pro+移动SIM卡
2.上网速度：在外70M，屋内30M
3.现象：屋里使用，有时会断网；过了1分钟左右又能自动连接上
4.要求：能稳定链接不断网
```

## 三 优化WiFi不掉线

### 3.1 先做这3步(80%问题直接解决)

Step 1：关闭自动休眠(最关键)

```
1、说明：很多人断网的“真凶”

2、操作：
-连接WiFi
-浏览器打开：192.168.8.1
-登录后台
-进入：高级设置 → WLAN → WLAN高级设置
-找到：「自动休眠 / WLAN休眠」
-改成：永不休眠
-保存 → 重启设备

3、效果：
1分钟自动恢复的问题基本消失
```

 Step 2：换摆放位置(立刻见效)

```
1、信号：你的情况“屋里30M + 断网”，典型就是信号问题

2、正确放法：
-窗边 / 阳台
-离地1米以上（书架/桌子）
-靠近使用区域

3、错误放法：
-房间角落
-地上
-被遮挡（柜子/墙后）

4、优先级仅次于关闭休眠
```

Step 3：锁定4G(避免反复切换)

```
1、操作：
-进入后台 → 网络设置
-设置为：仅4G（LTE Only）

2、原因：
-5G/4G来回跳 → 必断网
-4G更稳
```

### 3.2 WiFi稳定性优化(屋内断网重点)

Step 4：手动选WiFi信道(避免邻居干扰)

```
1、操作路径：
WLAN设置 → 高级设置 → 信道

2、推荐：
-2.4GHz：选 1/6/11
-5GHz：选 36/44

3、工具推荐：
WiFi Analyzer(看哪个信道最干净)
```

 Step 5：选对频段(关键)

|  场景  |  选择  |
| :----: | :----: |
| 隔墙多 | 2.4GHz |
| 同房间 |  5GHz  |

建议：

```
默认用 2.4GHz更稳
近距离再用5G
```

Step 6：远离干扰源

```
不要放在：
 -微波炉旁
 -冰箱旁
 -蓝牙设备密集处
```

### 3.3 移动信号优化(解决“间歇断网”)

Step 7：找最佳信号点(很重要)

```
1、用手机测试：
-打开移动数据
-看信号格数/测速

2、查看信号强度：
打开192.168.8.1，高级设置—>系统—>设备信息(SINR信号强度)

3、把设备放在：
-信号最强的位置（通常是窗边）
```

Step 8：基础重置操作

```
按顺序执行：
 -重启移动WiFi
 -拔插SIM卡
 -重启手机WiFi
 -忘记WiFi重新连接
```

Step 9：升级固件

```
登录后台 → 系统 → 更新
很多掉线是固件bug
```

### 3.4 进阶稳定方案(推荐做)

方案1：接一个路由器(强烈推荐)

```
1、连接方案：
移动WiFi → 路由器 → 手机/电脑

2、优点：
 -覆盖更强
 -不容易掉
 -多设备更稳
```

方案2：外置天线(如果支持)

```
效果最明显：
 -信号提升
 -减少基站切换
```

方案3：长期使用建议

```
-每月重启1次
-不用时关机
-避免高温环境
```

## 四 快速判断是哪种问题

|     现象     |     原因      |
| :----------: | :-----------: |
| WiFi在但没网 | 移动信号问题  |
|  WiFi直接断  | WiFi设置/干扰 |
|  1分钟恢复   | 休眠/基站切换 |

## 五 最终推荐

```
1.关闭休眠
2.放窗边 + 高处
3.锁定4G
4.固定信道(1/6/11)
```
