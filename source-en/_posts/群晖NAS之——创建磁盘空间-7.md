## 一 概述

```
本文介绍：
-从硬盘变成可操作文件的过程
-群晖中，存储结构分为三层：硬盘→存储池→存储空间(Volume)→共享文件夹
-本文以DSM 7+界面为例讲解
```

<!--more-->

## 二 步骤详解

### 2.1 创建存储池(Storage Pool)

```
1.打开 Storage Manager(存储管理器)。
2.去 存储 页面 → 右上 创建 → 创建存储池。
3.RAID类型：新手选 Basic(单盘简单)或 SHR(以后加盘方便)。
4.选硬盘(先擦除新盘：HDD/SSD → 动作 → 擦除 → 快速)。
5.描述起名(比如“电影10T-A”)，下一步 → 应用。
6.后台检查硬盘，几分钟到几小时。
```

### 2.2 创建存储空间(Volume，卷)

```
1.还在Storage Manager → 存储 → 创建 → 创建卷。
2.选刚才的存储池。
3.容量：点Max全用。
4.文件系统：选Btrfs！(推荐)
为什么？防位腐(电影默默坏掉能查)、支持快照(误删回滚)、定期scrub检查。ext4简单但没这些保护。
5.应用，几分钟OK。卷编号如volume3。
```

### 2.3 File Station创建文件夹(共享文件夹 + 子文件夹)

```
1.控制面板 → 共享文件夹 → 创建 → 创建共享文件夹。
 -名称：电影10T-A
 -位置：选你的Btrfs卷。
 -描述、权限默认或自定义。
 -应用。

2.打开 File Station，左边找到共享文件夹，右键 → 创建文件夹(子目录如“2026新片”“韩剧”)。
3.以后电脑映射网络盘，就能拖文件进去了。
```

## 三 补充

```
1.Btrfs卷建完，Storage Manager → 动作 → 创建快照计划(每周自动)。
2.每月跑数据清理(scrub)，查坏块。
3.多卷管理：一个卷放电影，一个放照片，互不干扰。

一步步做，新手半小时内就能用上NAS。创建时别慌，界面有向导。
```
