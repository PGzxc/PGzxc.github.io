## 二 布局适配建议

### 2.1 布局适配建议

|      场景       |                 推荐写法                  |
| :-------------: | :---------------------------------------: |
|    普通页面     |      .safeArea(SafeAreaType.SYSTEM)       |
| 视频 / 相册全屏 |       .safeArea(SafeAreaType.NONE)        |
|   刘海屏适配    |      .safeArea(SafeAreaType.CUTOUT)       |
|    键盘遮挡     |     .safeArea(SafeAreaType.KEYBOARD)      |
|  平板或折叠屏   | 动态获取 AvoidArea 并监听 avoidAreaChange |

### 2.2 示例

```
Stack() {
  Image($r('app.media.video_bg'))
  Column() {
    Text('视频播放中')
  }
  .safeArea(SafeAreaType.NONE)
}
```

## 三 状态栏/导航栏 适配与沉浸式控制

### 3.1 设置状态栏是否显示

```
const win = await window.getLastWindow(this.getContext())
await win.setWindowSystemBarEnable(['navigation']) // 只显示导航栏

隐藏全部系统栏（全屏沉浸式）：
await win.setWindowSystemBarEnable([]) 
```

### 3.2 设置状态栏颜色

```
await win.setWindowSystemBarProperties({
  statusBarColor: '#000000',
  navigationBarColor: '#000000',
})
```

## 四 折叠屏/平板 适配技巧

```
鸿蒙设备多样，建议：

-动态读取 AvoidArea，不要写死 dp 值；
-横屏时重新计算安全区；
-用 .safeArea() 适配多设备形态；
-窗口分屏模式（multi-window）下建议监听 avoidAreaChange。
```
