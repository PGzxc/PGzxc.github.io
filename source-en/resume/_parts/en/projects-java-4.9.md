### <font color=red>9. Java Projects</font>

#### 1—[TeaOrderAPI  ](https://github.com/PGzxc/TeaOrderAPI)

**Project Ownership** ：Personal Project (Private)
**Project Name**：TeaOrderAPI - Tea Beverage Order Management System
**Project URL**：https://github.com/PGzxc/TeaOrderAPI
**Platform Support**：Java Web Application
**Development Tools**：IntelliJ IDEA 2024.1 + Maven 3.9 
**Project Description**：A back-end service for tea beverage industry order management built on Spring Boot, providing unified RESTful API interfaces for mini-programs, apps, and management backends, implementing core business functions such as product management, seat management, shopping cart, and order processing.
**Functional Modules**：Category Management, Product Management, Seat Management, Shopping Cart Management, Order Management, File Upload
**Technical Highlights**：

- Building RESTful API interfaces based on Spring Boot 4.0.5 + Spring MVC
- Implementing data persistence based on Spring Data JPA, supporting automatic table structure generation
- Designing relational database based on MySQL 8.x, supporting utf8mb4 character set
- Implementing cache acceleration based on Redis to improve system response performance
- Implementing file upload functionality based on MultipartResolver, supporting image storage and access
- Simplifying entity class code based on Lombok to improve development efficiency
- Implementing hot reload during development based on Spring Boot DevTools to accelerate development iterations
- Ensuring code maintainability based on layered architecture design (Controller/Service/Repository/Entity)
