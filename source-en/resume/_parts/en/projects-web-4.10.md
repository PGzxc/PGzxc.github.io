### <font color=red>10. Web Frontend Projects</font>

#### 1—[tea-admin-panel  ](https://github.com/PGzxc/tea-admin-panel)

**Project Ownership** ：Personal Project (Private)
**Project Name**：TeaOrderAPI - Tea Beverage Order Management System
**Project URL**：https://github.com/PGzxc/tea-admin-panel
**Platform Support**：Modern Browsers (Chrome, Firefox, Safari, Edge)
**Development Tools**：Visual Studio Code + Vite 8.0 
**Development Language** ：Vue 3 + JavaScript
**Project Description**：A backend management system for tea beverage shops built on Vue 3 + Element Plus, integrated with Spring Boot backend services to implement core functions such as product management, order processing, seat monitoring, and data statistics. The system adopts a frontend-backend separation architecture, uses Axios for API calls, and supports business scenarios such as product image upload, real-time seat status updates, and order status transitions.
**Functional Modules**：

-  Dashboard ：Data overview, business metrics display
-  Product Management ：Product CRUD, image upload, category management 
-  Order Management ：Order list, status updates, detail viewing
-  Seat Management ：Seat grid view, status management, add/delete
-  System Settings ：Basic configuration, data statistics

**Technical Highlights**：

- Vue 3 Composition API based：Managing component state through ref, computed, watch and other APIs
- Element Plus Component Library：Building management interfaces using components such as tables, forms, dialogs, and buttons
- Axios HTTP Client：Encapsulating unified API calls, handling request/response interception and error handling
- Vue Router Routing Management：Implementing page navigation, route lazy loading, optimizing first screen loading performance
- Vite Build Tool：Utilizing ES Module for fast development and optimized production builds
- Pinia State Management：Managing cross-component shared state, implementing data persistence
- RESTful API Design：Following REST specifications, using HTTP methods to express operation semantics
- Frontend-Backend Separation Architecture：Vite proxy configuration solves cross-domain issues, enabling independent development and deployment
