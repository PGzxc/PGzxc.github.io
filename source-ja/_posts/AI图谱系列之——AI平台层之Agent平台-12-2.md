## 二 AI平台层

### 2.1 平台层理解

```
1.这一层可以理解为
“把模型、Agent、工具、推理能力全部打包成可用服务的平台”

2.本质上是：
AI能力 → 平台化 → 产品化 → 商业化
```

### 2.2 平台分类

```
- MCP平台
- Agent平台
- 应用平台
- 模型平台
- 推理平台
- Serverless AI平台
```

## 三 Agent平台(自动化执行层)

### 3.1 工具

```
1.工具
OpenClaw
```

### 3.2 核心能力

```
1.概述
任务 → 自动执行

2.包括：
-自动规划（Planning）
-自动执行（Action）
-工具调用（Tool Use）
-多Agent协作
```

### 3.3  注意

```
1.强大但风险高

2.需要：
-沙箱环境
-权限控制
-行为约束（Guardrails）
```
