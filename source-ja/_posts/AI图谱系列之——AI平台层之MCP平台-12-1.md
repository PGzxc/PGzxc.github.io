## 二 AI平台层

### 2.1 平台层理解

```
1.这一层可以理解为
“把模型、Agent、工具、推理能力全部打包成可用服务的平台”

2.本质上是：
AI能力 → 平台化 → 产品化 → 商业化
```

### 2.2 平台分类

```
- MCP平台
- Agent平台
- 应用平台
- 模型平台
- 推理平台
- Serverless AI平台
```

## 三 MCP平台(工具生态核心)

### 3.1 工具

```
1.工具
LobeHub

2.核心作用：
连接 AI 与 外部世界（工具/API/系统）
```

### 3.2 能力拆解

```
-MCP工具管理
-多工具调用（Tool Use）
-本地 / 远程工具接入
-Agent能力扩展
```

### 3.3  本质

```
MCP = AI的“USB接口”
所有能力（搜索 / 数据库 / 浏览器）都通过它接入
```
