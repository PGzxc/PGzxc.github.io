## 二 AI开发范式

### 2.1 总体理解

```
传统开发：人写代码 → 调API → 完成功能
AI开发：人描述目标 → AI规划 → AI执行 → 人做决策
```

### 2.2  本质变化

```
“编写代码” → “编排智能”
```

## 三 Computer Use Paradigm(屏幕操作范式)

### 3.1  核心思想

```
AI像人一样操作电脑
```

### 3.2 能力

```
-点击 / 输入 / 浏览网页
-操作软件（IDE / 浏览器）
-自动执行任务
```

### 3.3 代表能力

```
自动填表
自动爬数据
自动操作系统
```

### 3.4 风险

```
安全问题（误操作）
权限控制
```
