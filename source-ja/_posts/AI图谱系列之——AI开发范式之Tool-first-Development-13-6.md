## 二 AI开发范式

### 2.1 总体理解

```
传统开发：人写代码 → 调API → 完成功能
AI开发：人描述目标 → AI规划 → AI执行 → 人做决策
```

### 2.2  本质变化

```
“编写代码” → “编排智能”
```

## 三 Tool-first Development(工具优先)

### 3.1  核心思想

```
先定义工具，再让AI调用
```

### 3.2 示例

```
1.定义：
- 搜索工具
- 数据库工具
- 支付API

AI自动组合调用
```

### 3.3 本质

```
AI = 调度器（Orchestrator）
```
