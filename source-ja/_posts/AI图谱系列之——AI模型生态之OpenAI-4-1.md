## 三 OpenAI(闭源最强生态)

### 3.1 OpenAI 当前模型体系怎么划分？

```
1-通用模型（主力）
模型：GPT-4o/GPT-5.4（最新）

特点：
-多模态（文本/图像/音频）
-高质量生成
-支持长上下文（~1M）

2-推理模型（Reasoning）
模型：o1/o3/o3-pro

特点：
-强逻辑推理（数学/代码）
-支持 Chain-of-Thought（内部）

3-Thinking/Pro 模式（2026重点）

本质：
-Test-time Compute（推理时多算）
-更高成本 → 更高准确率

面试加分点：类似“推理深度调节”
```

### 3.2 GPT-5 相比 GPT-4 的核心提升？

```
1.提升
-上下文：128K → 1M
-推理能力显著增强
-原生 Computer Use（自动操作电脑）
-更强工具调用能力（Agent化）

2.面试关键词：
“Agent-ready 模型”
```

### 3.3 Codex 为什么被淘汰？

```
1.Codex已被：

2.原因：
-通用模型代码能力已足够强
-Agent能力更重要
```
