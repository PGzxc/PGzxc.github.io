## 二 AI理论基础/模型优化

### 2.1 什么是知识蒸馏？

```
1.概念
知识蒸馏是使用 大模型训练小模型。

2.结构：
Teacher Model
↓
Student Model

3.目标：
-模型压缩
-提高推理效率
```

### 2.2 什么是模型剪枝(Pruning)

```
1.概念
剪枝是 删除不重要的权重或神经元。

2.类型：
 -Weight Pruning
 -Neuron Pruning
 -Structured Pruning

3.效果：
 -减少参数
 -减少计算
 -提高速度
```

### 2.3 什么是模型量化(Quantization)？

```
1.概念
量化是降低模型参数精度。

2.例如：
FP32 → FP16 → INT8 → INT4

3.优势：
-减少显存
-提高推理速度
```

### 2.4 模型压缩(Model Compression)有哪些方法？

```
1.常见方法：
 -Knowledge Distillation
 -Quantization
 -Pruning
 -Low-Rank Decomposition

2.目标：
 -减少模型体积
 -提高推理速度
```

### 2.5 什么是稀疏激活？

```
1.概念
稀疏激活是 每次只激活部分模型参数。

2.例如 MoE：
总参数：1T
推理激活：50B

3.优势：
降低计算成本
扩大模型规模

4.代表模型：
DeepSeek
Mixtral
Qwen MoE
```
