## 二 机器学习

### 2.1 什么是 机器学习(Machine Learning)?

```
1.概念
Machine Learning(机器学习)是人工智能的一个子领域，
指计算机通过数据自动学习规律并进行预测或决策，而不需要显式编写规则。

核心思想是学习函数：y = f(x)
其中：
x = 输入数据
y = 预测结果
f = 模型


2.机器学习基本流程：
数据收集—>数据预处理—>模型训练—>模型评估—>模型部署

3.典型应用：
 -推荐系统
 -图像识别
 -语音识别
 -金融风控
 -搜索排序
```

### 2.2 机器学习(Machine Learning)与深度学习(Deep Learning)区别

1-概念

```
Deep Learning 是 Machine Learning 的一个子领域，主要使用深层神经网络。
```

2-对比

|   维度   | Machine Learning | Deep Learning |
| :------: | :--------------: | :-----------: |
| 特征工程 |   需要人工设计   |   自动学习    |
| 数据规模 |     中小规模     |    大规模     |
| 计算资源 |     CPU即可      |    GPU/TPU    |
| 模型结构 |     传统算法     |   神经网络    |
|   应用   |    预测、分类    | NLP、CV、语音 |

3-典型算法

```
1.机器学习：
Logistic Regression
Decision Tree
Random Forest
SVM

2.深度学习：
CNN
RNN
Transformer
```

### 2.3 什么是神经网络(Neural Network)?

```
1.概念
神经网络（Neural Network）是一种模仿人脑神经元结构的机器学习模型，由多个神经元连接组成。

2.基本结构：

Input Layer
Hidden Layers
Output Layer

3.神经元计算：

y = activation(w · x + b)

其中：
w = 权重
b = 偏置
activation = 激活函数

4.常见激活函数：
ReLU
Sigmoid
Tanh
Softmax
```

### 2.4 什么是监督学习(Supervised Learning)?

```
1.概念
监督学习是使用 带标签数据训练模型 的机器学习方法。

2.数据形式：(x, y)

其中：
x = 输入
y = 标签

3.目标：
学习映射关系：f(x) ≈ y

4.常见任务：
 -分类
 -回归

5.应用：
 -垃圾邮件检测
 -图像分类
 -信用评分
```

### 2.5 什么是无监督学习(Unsupervised Learning)？

```
1.概念
无监督学习使用 没有标签的数据，目标是发现数据内部结构。

2.数据：x

3.常见任务：
 -聚类
 -降维
 -异常检测

4.常见算法：
 -K-Means
 -DBSCAN
 -PCA
 -AutoEncoder

5.应用：
 -用户分群
 -异常检测
 -数据压缩
```

### 2.6 什么是强化学习(Reinforcement Learning)？

```
1.概念
强化学习是通过 与环境交互并获得奖励（Reward）来学习策略（Policy） 的机器学习方法。

2.核心组件：
 -Agent
 -Environment
 -State
 -Action
 -Reward

3.目标：
最大化累计奖励

4.应用：
 -AlphaGo
 -机器人控制
 -自动驾驶
 -LLM对齐（RLHF）
```
