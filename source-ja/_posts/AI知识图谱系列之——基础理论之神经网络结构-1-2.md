## 二 神经网络结构

### 2.1 CNN主要用于什么？

```
1.概念
CNN（Convolutional Neural Network）:卷积神经网络
是一种用于 图像处理 的神经网络。

2.核心思想：
 -卷积操作提取局部特征

3.核心结构：
 -Convolution Layer
 -Pooling Layer
 -Fully Connected Layer

4.应用：
 -图像分类
 -目标检测
 -人脸识别
 -医学影像
```

### 2.2 RNN有什么特点？

```
1.概念
RNN（Recurrent Neural Network）：循环神经网络
适合处理 序列数据。

2.特点：
当前状态依赖历史状态

3.公式：
h_t = f(x_t , h_(t-1))

4.应用：
 -语音识别
 -时间序列预测
 -文本生成

5.问题：
 -梯度消失
 -长依赖问题

6.解决方案：
 -LSTM
 -GRU
```

### 2.3 Transformer为什么取代RNN？

```
1.概念
Transformer 使用 Attention机制，具有以下优势：
 -支持并行计算
 -捕获长距离依赖
 -训练效率高
 -扩展性强

2.核心结构：
 -Self-Attention
 -Feed Forward Network
 -LayerNorm
 -Residual Connection

3.应用：
 -GPT
 -BERT
 -LLM
 -多模态模型
```

### 2.4 什么是Attention机制？

```
1.概念
Attention 是一种计算输入之间重要性权重的机制。

2.公式：

Attention(Q,K,V) = softmax(QK^T / √d) V

含义：
Q = Query
K = Key
V = Value

3.作用：
计算信息之间的相关性
```

### 2.5 什么是Self Attention？

```
1.概念
Self Attention 是 序列内部元素之间的注意力计算。

2.例如：

句子：The cat sat on the mat

模型可以学习：
cat ↔ sat
cat ↔ mat

3.优势：
捕获长距离依赖
```

### 2.6 什么是Cross Attention？

```
1.概念
Cross Attention 是 不同序列之间的注意力机制。

2.示例：
 -文本 → 图像
 -文本 → 文本

3.应用：
 -Encoder-Decoder
 -多模态模型
 -Stable Diffusion
```

### 2.7 什么是MoE？

```
1.概念
MoE 是一种 专家混合模型。

2.
结构：多个Expert网络+Gate Router
推理时只激活：Top-K Experts

3.优势：
 -参数规模大
 -计算成本低

4.代表模型：
 -Mixtral
 -DeepSeek
 -Qwen MoE
```
