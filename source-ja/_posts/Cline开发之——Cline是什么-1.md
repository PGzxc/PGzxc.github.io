## 二 Cline是什么

### 2.1 Cline官方

```
1.官网地址：https://cline.bot/
2.它是编程助手，集成于IDE(VSCode、JB IDE等)还可以是CLI
```

### 2.2 它可以

```
-写代码
-修改项目文件
-执行终端命令
-分析项目结构
-自动完成复杂开发流程
-并且所有操作 需要你确认（安全机制）
```

### 2.3 一句话理解

```
Cline = IDE 内的自动编程代理(Agent)
```

## 三 核心特点

```
-自主编码（Agent能力）
-可直接读写项目文件
-可执行终端命令
-支持浏览器操作（测试 Web）
-支持多模型（OpenAI/Claude/OpenRouter 等）
```

## 四 和 Copilot 的区别

1-对比

|   对比   |       Cline       |  Copilot   |
| :------: | :---------------: | :--------: |
|   类型   | Agent（主动执行） |  补全工具  |
|   能力   |  可操作文件+命令  | 只生成代码 |
|  自动化  |        强         |     弱     |
| 复杂任务 |       支持        |   不支持   |

2-结论

```
Cline = “AI开发助手” 
Copilot = “AI补全工具"
```
