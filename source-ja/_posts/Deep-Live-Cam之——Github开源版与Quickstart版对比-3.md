## 二 二者对比

### 2.1 本质区别

```
1.GitHub 版：
开源、免费、纯源码，需自己搭环境、装依赖、下模型。

2.Quickstart 版：
官网打包的 便携免安装版，内置 Python、CUDA、模型、虚拟摄像头，解压即用，但需要订阅付费。
```

### 2.2 核心对比

1-费用与授权

|   项目   |      GitHub 开源版       |        Quickstart 版         |
| :------: | :----------------------: | :--------------------------: |
|   费用   |    永久免费，MIT 开源    |     付费订阅制(按月/年)      |
|   源码   | 完全开放，可改可二次分发 | 闭源，不可修改、不可二次分发 |
| 商业使用 |       可(遵守 MIT)       |          需商业授权          |

2-安装与上手难度

|   项目   |                        GitHub 开源版                         |        Quickstart 版         |
| :------: | :----------------------------------------------------------: | :--------------------------: |
| 环境依赖 | 手动装：Python 3.10/3.11、Git、FFmpeg、CUDA、cuDNN、VS 运行库 |      全部内置，解压即跑      |
| 模型下载 |         手动去 HuggingFace 下 2 个模型，放到 models/         | 已打包内置，首次运行自动解压 |
| 启动方式 |            命令行：python run.py 或 run-cuda.bat             |   双击 exe/bat，图形化界面   |
| 上手门槛 |               中高(需要一点命令行 / 环境知识)                | 极低(纯小白也能 10 分钟开播) |

3-功能差异

|             功能             |         GitHub 开源版          |          Quickstart 版          |
| :--------------------------: | :----------------------------: | :-----------------------------: |
|           实时换脸           |              支持              |              支持               |
|     多人换脸(Many Faces)     |         支持(多人换脸)         |         支持(更早支持)          |
|     嘴巴遮罩(Mouth Mask)     |              支持              |       支持(可调参数更多)        |
|       面部增强 GFPGAN        |              支持              |      支持(内置，一键开关)       |
|    高清增强 GPEN 256/512     |           不支持(无)           |         支持(2.7+ 新增)         |
| 虚拟摄像头(OBS / 会议直接选) | 不支持(官方版无，需第三方工具) | 支持(内置虚拟摄像头，一键启用)  |
|        自动 CUDA 优化        |  不支持(需手动改代码 / 参数)   | 支持(自动识别 3060Ti，最优参数) |
|             界面             |          简洁原生 UI           |      美化 UI，更多快捷开关      |

4- 性能与速度(Win11 + 3060Ti)

```
1.GitHub 版
默认：Inswapper 128 + GFPGAN，约 8–12 FPS
需手动调 CUDA、关闭不必要增强，才能到 15 FPS 左右

2.Quickstart 版
出厂优化：自动用 CUDA、合理分辨率，约 12–18 FPS
关闭增强：稳定 20 FPS+（3060Ti 足够）
```

5-更新与支持

```
1.GitHub 版
更新慢：官网 Quickstart 新功能 滞后约 1–2 个月 才合入开源版
支持：社区 Discord、Issue，无官方一对一支持

2.Quickstart 版
更新快：新功能优先 60 天体验（如 GPEN、高级遮罩）
支持：官方 Discord + 邮件优先支持，问题响应快
```

6-隐私与安全

```
1.GitHub 版：本地运行、无联网校验，数据完全自己掌控

2.Quickstart 版：需联网激活订阅，官方可能收集匿名使用数据（隐私政策有写）
```

## 三 适合人群(怎么选)

### 3.1 选 GitHub 开源版

```
完全不想花钱
会一点电脑基础（装软件、改路径、用命令行）
想二次开发、改代码、自用定制
极度在意隐私，不想软件联网
```

### 3.2 选 Quickstart 版

```
纯小白，只想快速开播 / 视频通话换脸
想要 虚拟摄像头直接给 OBS/Zoom/Teams 用
想用最新功能（如 GPEN 高清增强、精细遮罩）
不想折腾环境，追求 开箱即用 + 稳定
3060Ti 显卡，想直接拿到 最优性能
```
