## 二 工作流实战

### 2.1 团队最佳实践流程

```
1. 安装 ECC GitHub App
2. /ecc-tools analyze
3. Review PR（生成规范）
4. 本地加载 ecc-universal
5. 使用 skills + agents
6. 开启 continuous learning
7. 使用 AgentShield 做安全扫描
```

### 2.2 日常开发流程

```
需求 → /plan
开发 → /tdd
提交 → hooks自动检查
PR → ECC自动分析
上线 → AgentShield安全扫描
```
