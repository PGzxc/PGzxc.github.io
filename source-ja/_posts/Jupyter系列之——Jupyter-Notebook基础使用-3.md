## 二 Notebook 结构

```
一个 Notebook 由多个 Cell 组成：

1.Code Cell
print("Hello Jupyter")

2.Markdown Cell
# 标题
**加粗**
```

## 三 快捷键(高频)

```
- Shift + Enter：执行
- A / B：插入 Cell
- M：转 Markdown
- Y：转 Code
```

## 四 执行模型

```
- 按 Cell 执行（非线性）
- 状态由 Kernel 维护

常见坑：

- 顺序错乱导致结果错误
```
