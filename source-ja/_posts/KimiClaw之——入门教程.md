## 二 KimiClaw是什么？

### 2.1 官网入口

```
https://www.kimi.com/bot
```

### 2.2 KimiClaw 是什么

```
1. 一句话理解：
KimiClaw = “可配置 + 可自动执行”的 AI Agent（智能体）

2. 它不是普通聊天机器人，而是：

- 可自定义人格（像一个员工）
- 可安装技能（像插件）
- 可自动执行任务（像脚本/定时器）
- 可长期记忆（像数据库）

3. 本质是：
Kimi + OpenClaw Agent 框架的云端版本
```

## 三 核心能力及使用模式

### 3.1 核心能力(理解它能干嘛)

```
一、智能体（Agent）
- 不只是问答，而是“执行任务”
- 支持：分析 / 写作 / 编程 / 自动化

类似：
- ChatGPT + 自动执行
- Claude + Workflow


二、技能系统（Skills）

内置技能库（ClawHub）：
- 数据分析
- 信息汇总
- 编程助手
- 文档处理

不用自己写 Prompt，直接用

三、定时任务（自动化）

可以让 AI 自动干活：每天 9:00 输出 AI 行业新闻（3条 + 风险提示）
本质： = AI + cron 定时任务 

四、长期记忆 + 人设

你可以定义：
- 名字
- 风格（严谨 / 幽默）
- 输出格式

类似：
你是一个高级架构师，输出必须包含：
1. 结论
2. 原因
3. 可执行步骤
```

### 3.2 使用模式(2种)

```
模式1：一键创建（推荐）

零门槛：
- 无需服务器
- 无需 API Key
- 自动配置模型（Kimi K2.5）

适合：
- 新手
- 快速体验 

模式2：连接 OpenClaw（进阶）

本地/自托管 Agent 接入

可以做到：
- 本地运行 AI
- 接入 Telegram / Slack
- 完全自定义流程

适合：
- 开发者
- 私有部署
```

## 四 5分钟快速上手

### 4.1 打开入口

```
https://www.kimi.com/bot
```

### 4.2 创建 Claw

```
点击：「创建 / 开始使用」
```

### 4.3 配置智能体

```
设置 3 个关键点：

1.人设（最重要）

你是一个技术专家，回答必须：
- 简洁
- 有结构
- 可执行

2.技能（可选）

选择：
- 信息总结
- 编程
- 数据分析

3. 输出格式（关键）

输出格式：
1. 结论
2. 步骤
3. 示例
```

### 4.4 直接对话使用

```
示例：
帮我分析 React Native TurboModule 实现方案
```

### 4.5 设置自动任务

```
每天 10 点：
总结 AI 编程工具更新（3条 + 趋势分析）
```

## 五 进阶玩法(开发者重点)

### 5.1 远程控制 Agent(强)

```
连接 OpenClaw 后：

你可以：
- 用 Kimi 控制本地程序
- 自动执行脚本
- 接入 IM（Telegram等）

本质：= ChatGPT + SSH + 自动化 
```

### 5.2 工作流自动化

```
典型场景：

1.AI 产品经理
输入需求 → 自动拆解 → 输出 PRD

2.AI 开发助手
需求 → 生成代码 → 优化 → Review
```

### 5.3 AI + RAG(知识库)

```
Kimi 支持长文本 +资料分析：
- 论文
- API 文档
- 项目代码

可做：
- 私有知识库助手 
```
