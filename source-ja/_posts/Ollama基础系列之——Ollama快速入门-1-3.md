## 二 模型来源

```

https://ollama.com/library

2.支持：
-Llama
-Qwen
-Mistral
-DeepSeek
```

## 三 常用命令

```
ollama list        # 查看模型
ollama pull qwen   # 下载模型
ollama run qwen    # 运行模型
ollama rm qwen     # 删除模型
```

## 四 运行第一个模型

```
1.指令
ollama run llama3

2.说明：
自动下载 + 运行
```
