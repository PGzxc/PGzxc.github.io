## 二 有关VPS

### 2.1 什么是 VPS？

```
VPS = 远程服务器（你自己的“云电脑”）

你可以把它理解为：
一台 24 小时开机、在国外机房运行的 Linux 电脑
```

### 2.2 VPS 在代理体系中的作用

```
1.中转服务就是VPS

你 → 中转服务器 → 目标网站

2.实际结构：

你的电脑
   ↓
本地代理（Clash）
   ↓
VPS（运行代理服务）
   ↓
互联网
```

### 2.3 为什么必须自己买 VPS？

```
1.很多新手会问：能不能用别人提供的？
不建议

2.原因：
- 不稳定：随时失效
- 不安全：流量被监控
- 不可控：随时被封

3. 自建的优势
 -独享资源
 -可控
 -更稳定
 -更安全
```

## 三 服务商及购买

### 3.1 推荐服务商

```
DMIT：https://www.dmit.io/
```

### 3.2 为什么推荐它？

```
1.在开发者圈比较常见

2.优点：
-CN2 优化线路（对亚洲友好）
-延迟低（特别是香港 / 日本）
-稳定性高

```

### 3.3 怎么买 VPS

```
1.选择配置（新手推荐）-最低配置(完全够用（甚至多人用）)：
-CPU：1核
-内存：1GB
-硬盘：20GB
-带宽：1TB/月

2. 选择地区（很重要）
地区	   推荐指数	     特点
🇭🇰 香港	  5颗星	    延迟最低
🇯🇵 日本	   4颗星	      稳定
🇺🇸 美国	   3颗星	      带宽大

新手建议：优先选：香港 或 日本

3.操作系统选择

一定选：Ubuntu 22.04（推荐）
原因：教程多/兼容性最好

4.获取服务器信息(购买后你会得到)：

-IP地址：1.2.3.4
-用户名：root
-密码：xxxx
-端口：22
```

## 四 连接教程

### 4.1 连接 VPS

```
方式1：Windows（推荐）

工具：
Termius（最简单）
MobaXterm（功能强）

方式二：命令行（通用）
ssh root@你的IP

第一次连接会提示：
Are you sure you want to continue connecting (yes/no)?

输入：yes
```

### 4.2 连接成功后你会看到什么？

```
1.显示内容
root@vps:~#

2.说明：
-你已经进入服务器
-可以开始操作
```

### 4.3 新手必须做的 3 件事(安全 + 稳定)

```
1.更新系统
apt update && apt upgrade -y

2.设置时区（可选）
timedatectl set-timezone Asia/Singapore

3.开启防火墙（重要）
apt install ufw -y
ufw allow 22
ufw enable
```

## 五 常见问题

### 5.1 问题1：连不上服务器

```
原因：
-IP 错误
-防火墙拦截
-端口不是 22
```

### 5.2 问题2：速度很慢

```
原因：
-地区选错（选了欧洲）
-线路差
```

### 5.3 问题3：服务器被封

```
1.原因：
-使用不安全协议
-暴露端口

2.后面我们会用：
Xray + Reality 解决
```
