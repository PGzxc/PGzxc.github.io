## 四 实战部署

### 4.1 WireGuard 部署

```
Step 1 安装：

apt install wireguard -y

Step 2 生成密钥：

wg genkey | tee privatekey | wg pubkey > publickey

Step 3 配置服务端：

[Interface]
Address = 10.0.0.1/24
PrivateKey = 服务端私钥
ListenPort = 51820

[Peer]
PublicKey = 客户端公钥
AllowedIPs = 10.0.0.2/32


Step 4 启动：

wg-quick up wg0
```

### 4.2 客户端配置(示例)

```
1.示例：
[Interface]
PrivateKey = 客户端私钥
Address = 10.0.0.2/24

[Peer]
PublicKey = 服务端公钥
Endpoint = VPS-IP:51820
AllowedIPs = 0.0.0.0/0

2.含义：
所有流量 → 走 VPN
```

### 5.2 零信任网络(高级概念 )

1-什么是零信任？

```
核心思想：默认不信任任何设备
```

2-在你的架构中体现:

```
- 每个设备单独认证
- 每个连接加密
- 不暴露内网
```

3-实现方式

|   技术    |   作用   |
| :-------: | :------: |
| WireGuard | 身份认证 |
|   Xray    | 流量控制 |
|  防火墙   | 权限限制 |
