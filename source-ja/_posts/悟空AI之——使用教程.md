## 二 悟空AI

### 2.1 概念

```
悟空是钉钉推出的 AI 工作平台，
通过对话式交互帮你高效完成日程、文档、审批、会议等各类工作任务。

它不只是聊天助手，更是你的行动搭档——能直接操作钉钉中的文档、日程、审批等应用，
把指令变成实际行动。
```

### 2.2 核心优势

```
1.从对话到行动：一句话调动多个应用，真正帮你把事情做完
2.钉钉深度集成：手机、平板、电脑多端同步，随时随地掌控工作
3.技能生态扩展：丰富的预置技能，还可上传自定义技能
4.长期记忆能力：越用越聪明，记住你的偏好和工作上下文
```

## 四 核心功能详解

### 4.1 技能中心

```
1.技能中心：

是悟空的核心能力库，提供各类预置技能供你调用。

2.官方精选技能

- 设计类：Logo 设计、海报生成、UI 方案建议
- 猎头类：简历筛选、人才匹配、面试问题生成
- 财务类：报表分析、预算规划、税务咨询
- 开发类：代码生成、Bug 排查、技术方案设计
- 文档处理：周报生成、会议纪要、合同审查
- 数据分析：数据清洗、图表制作、趋势预测

3.使用方式

在对话框中输入任务描述，悟空会自动匹配相应技能。例如：

- "帮我生成一份销售周报"
- "分析这份 Excel 数据的趋势"
- "设计一个产品 Logo"
```

### 4.2 定时任务

```
1.概念：
设置自动化任务，让悟空定时为你执行重复性工作。

2.常见场景

- 行业资讯汇总：每日自动抓取指定领域的最新新闻
- 工作周报生成：每周五自动聚合本周工作内容生成周报
- 数据定期备份：定时备份重要文档和表格
- 提醒事项推送：定时发送待办事项提醒

3.配置步骤

- 进入"定时任务"页面
- 点击"新建任务"
- 选择任务类型和执行频率
- 设置触发时间（支持 cron 表达式）
- 配置输出内容和通知方式
```

### 4.3 模型配置

悟空提供多种模型选项，满足不同场景需求。

1-模型类型

| 模型类型 | 适用场景 | 特点 |
|:-------:|:-------:|:----:|
| 自定义模型 | 特定领域任务 | 可上传自有模型，适配专业场景 |
| 通用模型 | 日常办公任务 | 平衡性能和成本，适合大多数场景 |

2 -响应等级

| 等级 | 速度 | 精度 | 算粒消耗 | 推荐场景 |
|:---:|:----:|:---:|:-------:|:-------:|
| 标准 | 快 | 基础 | 低 | 简单查询、快速回复 |
| 高级 | 中等 | 较高 | 中 | 文档生成、数据分析 |
| 旗舰 | 较慢 | 最高 | 高 | 复杂决策、创意任务 |

3-选择建议

- 日常对话 → 标准响应 + 通用模型
- 报告撰写 → 高级响应 + 通用模型
- 战略规划 → 旗舰响应 + 自定义模型（如有）

## 六 最佳实践

```
- 自定义技能沉淀：将常用工作方法固化为专属技能
```

```
好的提示词能获得更好的结果：

1.推荐写法：

1. 各产品线销售额对比
```

## 七 常见问题

### 7.1 算粒不够用怎么办？

```
- 检查是否有不必要的定时任务在运行
- 降低非关键任务的响应等级
- 联系管理员申请更多算粒配额
```

### 7.2 如何上传自定义技能？

```
1. 进入技能中心
2. 点击"我的技能" → "新建技能"
3. 按照技能模板填写配置
4. 上传技能脚本或工作流文件
5. 测试验证后发布
```

### 7.3 定时任务不执行怎么办？

```
- 检查任务是否处于"启用"状态
- 确认算粒余额充足
- 查看执行日志定位错误原因
- 检查触发时间设置是否正确
```

### 7.4 悟空能访问外部系统吗？

```
- 支持通过 API 连接器访问授权的外部系统
- 需要在技能配置中设置认证信息
- 部分系统可能需要管理员审批
```

## 八 进阶技巧

### 8.1 长期记忆利用

```
悟空会记住你的工作习惯和偏好，主动利用这一特性：

- 固定格式的文档只需说明一次，后续会自动套用
- 常用数据源会被记录，无需重复指定
- 工作上下文会在会话间保持连贯
```

### 8.2 多端协同

```
- 电脑端：处理复杂文档和数据分析
- 手机端：随时发起对话、查看进度、接收通知
- 平板端：适合审阅和批注文档
```

### 8.3 团队协作

```
虽然悟空定位为"个人 AI 搭档"，但也可用于团队协作：

- 共享技能给团队成员
- 协作编辑同一份文档
- 通过悟空分配和跟踪任务
```
