## 三 施工步骤(详细且不会返碱)

### 3.1 步骤 1：铲除旧墙皮(关键)

```
1、返碱墙面一定要铲到“看不到粉化层为止”，至少做到：
-松动的腻子 → 全部铲掉
-起皮区域 → 全部铲干净
-若墙皮“粉粉的”，必须铲到硬层

2、说明：做得不好返碱100%回来。
```

### 3.2 步骤 2：霉菌处理 + 返碱处理

1、 喷除霉剂(或84稀释)

```
-喷上
-等 5～10 分钟
-用刷子清洗
-再清水擦一遍

作用：杀霉菌根
```

2、草酸/柠檬酸擦拭返碱白粉

```
1、配方：
-草酸：水 = 1 : 10
-柠檬酸粉：水 = 1 : 10

2、操作
-擦一次即可
-能化掉表面的碱分。
```

### 3.3 步骤 3：彻底干燥(最重要步骤之一)

```
1、墙面必须做到：
-用手摸是干的
-没有冰凉感
-潮气明显减弱

2、说明：
如果潮气没排出去，后面做什么都没用。

3、建议：
-电风扇吹 12～24 小时
-或开窗通风干 1 天
```

### 3.4 步骤 4：做墙体防水(JS → K11)

1、JS 防水底层渗透封闭

```
1、操作
-JS 加水搅拌
-用毛刷刷 整面墙

2、作用：
-封闭毛细孔
-阻止墙体水汽继续往室内冒
-为 K11 做粘结层

3、说明：
等待 1～2 小时。
```

2、K11 灰浆型防水(加固)

```
1、刷 1～2 遍

2、刷法：
第一遍横刷
第二遍竖刷（十字交叉）

3、作用：
-阻断湿气
-阻断外墙渗透
-防止返碱再次出现
```

### 3.5 步骤 5：抗碱封闭底漆(核心)

```
1、返碱出现的真正原因：
水泥中的碱分被水带到表面

2、抗碱底漆作用：
-化学封闭
-阻断碱迁移
-防止乳胶漆起皮粉化

3、说明：刷 1 遍即可。
```

### 3.6 步骤 6：修补找平(腻子)

```
1、根据墙面情况：
-小坑 用腻子粉修补
-大坑、砖缝裂缝 用修补砂浆

2、说明：干燥后打磨平整。
```

### 3.7 步骤 7：涂防霉墙面漆(面漆)

```
建议：
-面漆刷 2 遍
-每遍间隔 2 小时
-尽量选择耐水防霉级别
```

## 四 推荐方案(不会返碱)

### 4.1 最强＋最稳方案(强烈推荐)

```
1）铲墙皮 → 深度除霉 → 返碱处理
↓
2）JS 防水（整面）
↓
3）K11 防水（至少 1 遍）
↓
4）抗碱底漆（必须）
↓
5）腻子找平
↓
6）防霉墙面漆
```

### 4.2  效果

```
-不再返碱
-不再结霉
-墙面硬实，不粉化
-3～5 年都不会再次坏
-外墙未完全修复也能撑住
```
