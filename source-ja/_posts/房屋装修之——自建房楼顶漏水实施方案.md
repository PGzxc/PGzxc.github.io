## 二 材料清单(按“水泥面”精准匹配)

### 2.1 裂缝修补材料(必备)

|             材料             |               用途               |   预算   |
| :--------------------------: | :------------------------------: | :------: |
|       堵漏王(快干)5kg        | 修补裂缝、坑洞、起砂、落水口边缘 | 40–60 元 |
| 水泥基渗透结晶修补砂浆(可选) |      裂缝第二次封固，更耐久      | 25–40 元 |

备注：水泥面首选堵漏王，不用环氧，因为裂缝多为“表层温度裂”，不需结构加固。

### 2.2 大面积防水材料(主材)

|                材料                | 用量(30–40㎡顶面) |    预算    |
| :--------------------------------: | :---------------: | :--------: |
| JS 双组份柔韧防水涂料(1 套20–25kg) |       1 套        | 110–150 元 |
|         聚酯布/网格布(1卷)         |  对整个楼顶增强   |  40–60 元  |

备注：JS + 网格布是水泥面最耐用组合，兼具柔性与耐晒，不容易因热胀冷缩开裂

### 2.3 辅材与工具

|     材料     |    用途    | 预算  |
| :----------: | :--------: | :---: |
| 滚筒、伸缩杆 | 大面积涂刷 |  20   |
| 搅拌桶、刮板 |  必备工具  |  20   |
| 清洁剂/丙酮  |  清理油污  | 10–20 |
|   防水胶带   | 落水口加强 |  10   |

备注：综合预算：350~450 元左右(自施工版)

## 三 终极施工流程(按你的“水泥面”专业版)

按顺序做，寿命可以达到 6–10 年。

### 3.1 查找裂缝 + 清理基层

```
1、说明：非常关键！

2、做法：
-扫掉灰尘
-用铁刷或吹风枪清理浮沙
-油污用丙酮擦掉
-把裂缝用螺丝刀或锤子敲开：扩成 1–1.5 cm 深的“V 字槽”

3、备注：这样堵漏王才能抓牢，不会脱落。
```

### 3.2 裂缝修补(堵漏王)

```
1、场景：适用于水泥面的最强组合。

2、操作：
按 1:0.3 加水搅拌
调成“糊状”
打湿裂缝槽
把堵漏王用力压进去
抹平，等待 20 分钟硬化

3、适用部位：
-所有裂缝
-起砂/坑洞
-落水口四周
-阴角、女儿墙底部

3、备注：
堵漏王对水泥粘结强，抗负压能力强，户外非常稳！
```

### 3.3 第一遍 JS 防水(横向薄涂)

```
1、目的：封毛细孔 + 提供粘结层

2、操作：
-JS 双组份搅拌均匀
-用滚筒薄薄刷一遍
-不能太厚
-要等 2 小时初干
```

### 3.4 铺网格布(增强，水泥面必须)

```
1、说明：
水泥面易热胀冷缩导致裂缝，所以网格布很关键。

2、铺设方式：
-JS 第一遍未干前直接铺
-用刮板压平
-不留气泡、皱褶
-超过 3cm 的地方重叠

3、重点部位：
-裂缝全覆盖
-楼顶全部（推荐）
-女儿墙上翻 20–30cm
-落水口周边

4、备注：
有了网格布，JS 的柔韧性才能发挥最大效果。
```

### 3.5 第二遍 JS(纵向厚涂)

```
1、额度：比第一遍更厚，约 1mm

2、刷法：
-滚筒来回多刷几次
-与第一遍方向交叉（横 + 纵）
-彻底覆盖网格布
```

### 3.6 第三遍 JS(可选，但非常建议)

```
1、说明：
这一步能让寿命从 5 年 → 延长到 7–10 年。

2、刷法随意，多刷重点部位：
-女儿墙
-落水口
-所有可能积水区
```

### 3.7 落水口加强处理(关键区)

```
1、处理
落水口内壁刷 2–3 遍 JS
落水口口沿贴一圈防水胶带
如有坑洼先用堵漏王补平

2、备注：
90% 农村楼顶漏水出在落水口。
```

### 3.8  养护

```
-24 小时不可淋雨
-48 小时不踩踏
-完全固化 3 天后防水效果最好
```

## 四 为什么最终给你选这套组合

### 4.1 你家是“水泥楼面”，而不是瓷砖或卷材

```
水泥面最容易出现温度裂缝 → 弹性防水 +加布是顶配。
```

### 4.2 裂缝属于非结构性

```
无需环氧树脂灌缝（结构裂缝才适用）。
```

### 4.3 自施工最友好

```
这套组合你自己完全能做，不需要灌浆机、封缝枪等专业设备。
```

### 4.4 价格低、寿命高、材料好买

```
农村建材店都有 JS、堵漏王、网格布。
```

### 4.5 耐晒抗雨，适合裸露楼顶

```
JS+网格布是一种“柔性防水膜”，能跟随热胀冷缩移动。
```
