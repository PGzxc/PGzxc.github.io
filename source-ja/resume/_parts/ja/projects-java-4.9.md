### <font color=red>九 Javaプロジェクト</font>

#### 1—[TeaOrderAPI  ](https://github.com/PGzxc/TeaOrderAPI)

**プロジェクト所属** ：個人プロジェクト(プライベート)
**プロジェクト名**：TeaOrderAPI - 飲茶注文管理システム
**プロジェクトURL**：https://github.com/PGzxc/TeaOrderAPI
**プラットフォームサポート**：Java Webアプリケーション
**開発ツール**：IntelliJ IDEA 2024.1 + Maven 3.9 
**プロジェクト説明**：Spring Bootをベースに構築された飲茶業界の注文管理バックエンドサービスで、ミニプログラム、アプリ、管理バックエンドに統一されたRESTful APIインターフェースを提供し、商品管理、座席管理、ショッピングカート、注文処理などのコアビジネス機能を実現します。
**機能モジュール**：カテゴリ管理、商品管理、座席管理、ショッピングカート管理、注文管理、ファイルアップロード
**技術的ハイライト**：

- Spring Boot 4.0.5 + Spring MVCに基づくRESTful APIインターフェースの構築
- Spring Data JPAに基づくデータ永続化の実装、自動テーブル構造生成のサポート
- MySQL 8.xに基づくリレーショナルデータベースの設計、utf8mb4文字セットのサポート
- Redisに基づくキャッシュ加速の実現、システム応答パフォーマンスの向上
- MultipartResolverに基づくファイルアップロード機能の実現、画像の保存とアクセスのサポート
- Lombokに基づくエンティティクラスコードの簡素化、開発効率の向上
- Spring Boot DevToolsに基づく開発期間のホットリロードの実現、開発イテレーションの加速
- 階層化アーキテクチャ設計(Controller/Service/Repository/Entity)に基づく、コードの保守性の保証
