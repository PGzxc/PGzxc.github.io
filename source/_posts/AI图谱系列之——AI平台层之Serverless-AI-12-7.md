## 二 AI平台层

### 2.1 平台层理解

```
1.这一层可以理解为
“把模型、Agent、工具、推理能力全部打包成可用服务的平台”

2.本质上是：
AI能力 → 平台化 → 产品化 → 商业化
```

### 2.2 平台分类

```
- MCP平台
- Agent平台
- 应用平台
- 模型平台
- 推理平台
- Serverless AI平台
```

## 三 Serverless AI

### 3.1 工具

```
1.工具
-Modal
```

### 3.2 核心思想

```
GPU按需启动
自动扩缩容
任务级调度
```

### 3.3  能力

```
一键部署模型
按调用计费
支持多种模型（图像/音频/LLM）
```

### 3.4 对比传统

|    方式    |        特点         |
| :--------: | :-----------------: |
|  传统部署  |  成本高 / 运维复杂  |
| Serverless | 按需付费 / 自动扩展 |
