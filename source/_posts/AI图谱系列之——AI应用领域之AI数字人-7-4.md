## 三 AI数字人(虚拟人/Agent形态)

### 3.1 什么是AI数字人？

```
1.概念
AI数字人 = 虚拟形象 + 语音 + 动作驱动

2.技术组成：
-TTS
-语音驱动口型
-视频生成
```

### 3.2 主流平台?

```
HeyGen
D-ID
Synthesia
Hour One
```

### 3.2 核心能力

```
文本 → 视频（虚拟人播报）
语音驱动口型（Lip Sync）
多语言支持
虚拟形象生成
```

### 3.3 技术组成(面试重点)

```
LLM（大脑）
TTS（声音）
CV（人脸驱动）
动画系统（动作）
```
### 3.4 应用场景

```
AI主播/电商带货/电商直播
AI客服/企业客服
教育视频/教育培训
```

### 3.5 技术难点

```
1. 表情自然度
2. 口型同步
3. 实时性
4. 情感表达
```

### 3.6  一句话总结

```
AI数字人 = LLM + 语音 + 视觉驱动
```
