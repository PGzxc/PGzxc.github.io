### 3.2 IDE型（AI增强开发环境）

```
1.IDE
-Cursor
-Continue

2. 特点：
-类似 VSCode
-内嵌 AI Agent
-支持代码级上下文理解
```

### 3.3 Agent型（自动开发系统）

```
1.Agent型
-OpenDevin
-Smol Developer
-Manus

2.特点：

输入需求 → 自动：
- 拆任务
- 写代码
- 运行
- 修复

3.本质：AI软件工程师
```

### 3.4 工具链型（企业级架构）

```
1.工具链
-LangGraph Studio
-Dify

2.用于：
-搭建复杂Agent系统
-多模型调度
-多工具集成
```

### 3.5 前沿/生态工具

```
1、前沿/生态工具
-Google Antigravity
-Google ADK
-mobile-mcp

2.趋势方向：
-MCP生态（Agent标准化）
-移动端Agent
-多端协同
```

## 四 总结

### 4.1 一句话理解

```
AI开发工具正在经历三阶段演进：

Copilot（辅助）
   ↓
Agent（半自动）
   ↓
Autonomous Dev（全自动开发）
```

### 4.2 技术分层

```
应用层：
- Cursor / Dify / Flowise

编排层：
- LangGraph

能力层：
- LLM / RAG / Tool

基础层：
- 推理框架（vLLM等）
```

### 4.3 2026开发趋势

```
1. IDE → AI IDE（Cursor时代）
2. 编程 → Agent编程
3. 开发者 → AI指挥者
4. 本地模型IDE（隐私 + 成本）
5. MCP 打通工具生态
```
