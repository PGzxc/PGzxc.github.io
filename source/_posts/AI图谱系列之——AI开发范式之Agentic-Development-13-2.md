## 二 AI开发范式

### 2.1 总体理解

```
传统开发：人写代码 → 调API → 完成功能
AI开发：人描述目标 → AI规划 → AI执行 → 人做决策
```

### 2.2  本质变化

```
“编写代码” → “编排智能”
```

## 三 Agentic Development(Agent驱动开发)

### 3.1 理解

```
AI不再是工具，而是“开发者”
```

### 3.2 核心流程

```
需求 → Agent拆解 → 执行 → 修复 → 迭代
```

### 3.3 关键能力

```
Planning（任务拆解）
Tool Use（调用工具）
Memory（上下文记忆）
Reflection（自我修正）
```

### 3.4 本质

```
软件开发 = 多Agent协作系统
```

### 3.5 应用

```
自动写项目
自动测试
自动部署
```
