## 三  本地推理(边缘/离线)

### 3.1 本地推理方案有哪些？

1-llama.cpp(最经典)

```
1.说明：最主流 CPU 推理框架

2.特点：
-支持 GGUF
-可在 CPU / Mac / 嵌入式运行
```

2-ExLlama(GPU优化)

```
1.特点
-专为 GPU 推理优化
-支持量化模型（4bit / 8bit）

2.优势
-比 llama.cpp 更快（GPU场景）
```

3-MLC-LLM(移动端)

```
1.核心能力
-iOS / Android 本地运行
-WebGPU / Metal 加速

2.使用场景
-手机AI
-边缘设备
```

### 3.2 本地推理 vs 云推理？

1-对比

| 维度 |  本地  |  云  |
| :--: | :----: | :--: |
| 延迟 |   低   |  中  |
| 成本 | 一次性 | 持续 |
| 隐私 |   高   |  低  |
| 能力 |  有限  |  强  |

2-总结

```
“云模型负责能力，本地模型负责隐私和低延迟”
```

## 四 本地推理总结

```
llama.cpp   → CPU首选
ExLlama     → GPU本地推理
MLC-LLM     → 手机端AI
```
