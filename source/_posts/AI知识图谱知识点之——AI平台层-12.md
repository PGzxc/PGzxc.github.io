## 二 AI平台层

### 2.1 核心平台一句话定位

```
1.LobeHub
MCP 协议工具管理、多 Agent 控制面板、插件化平台。

2.OpenClaw
开源 Agent 平台，多模型、本地运行、社区生态最火。

3.Dify
低代码 LLM 应用平台，RAG + Agent + 工作流一站式搭建。

4.Hugging Face
全球最大模型托管、数据集、推理部署中心，开源模型事实标准。

5.Groq / Fireworks / Together
高速推理云平台，超低延迟、高吞吐，主打极速调用。

6.Replicate
AI 模型一键运行、调用，开箱即用，多模态模型最丰富。

7.Modal
Serverless AI 部署平台，代码一键上云，自动扩缩容、按量计费。
```

### 2.2 平台层整体作用

```
-提供模型托管、推理加速、应用搭建、Agent 编排、工具集成
-让开发者不用管底层基建，专注业务逻辑
-是连接「模型 → 开发 → 上线」的云端中枢
```

## 三 总结

### 3.1 一句话总结

```
-应用搭建：Dify
-Agent/MCP：LobeHub、OpenClaw
-模型开源大本营：Hugging Face
-极速推理：Groq、Fireworks、Together
-一键跑模型：Replicate
-Serverless 部署：Modal
```
