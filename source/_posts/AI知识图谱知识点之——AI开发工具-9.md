## 二 AI开发工具

| :------------: | :----------------: | :----------------------------------------------------------: | :-----------------------------------: |

| GitHub Copilot |   生态标杆副驾驶   |           GitHub 深度集成；安全扫描强；多 IDE 兼容           |  日常补全、开源协作、团队标准化开发   |

### 2.2 Agent 开发工具(2026 工作流标配)

|       工具       |     类型     |                2026 核心能力                |               适用人群                |
| :--------------: | :----------: | :-----------------------------------------: | :-----------------------------------: |
| LangGraph Studio |  工作流编排  | 状态机、多 Agent 协作、人类在环、检查点记忆 |   开发复杂 Agent、需强可控与可维护    |
|     Flowise      | 可视化无代码 | 拖拽搭建、一键导出 API、本地部署、MIT 开源  |    非开发、快速原型、隐私敏感场景     |
|       Dify       |  低代码平台  |     可视化编排、RAG 一体化、企业级权限      | 快速落地产品、需 API 化输出、团队协作 |
|    Jupyter AI    | 科研开发增强 |      单元格级 AI 辅助、代码与文档协同       |    数据分析、科研原型、交互式开发     |

选型一句话：LangGraph 做复杂可控，Flowise 做无代码快速，Dify 做企业落地。

### 2.3 Agent IDE(三大阵营)

```
1）IDE 型（基于编辑器增强）
-Cursor：编码与 Agent 一体，PlanMode 强，适合全栈开发
-Continue：VS Code 插件，无缝迁移，轻量高效

2）Agent 型（自主执行专家）
-OpenDevin：自主规划执行，适合端到端项目
-Smol Developer：轻量自主 Agent，适合快速原型
-Manus：企业级多 Agent 平台，适合团队协作

3）工具链型（工作流集成）
-LangGraph Studio：多 Agent 编排核心，状态管理强
-Dify：低代码可视化，快速生成 API 与应用

4）其他精选
-Trae / Qoder：中文场景优化，Builder 模式快速生成项目
-Kiro / opencode / Codebuddy：轻量 IDE，适合移动端与快速开发
-mobile-mcp：移动端 MCP 工具，适配边缘部署
-Google Antigravity / ADK：Google 原生工具链，适合安卓生态
-ClawdBot：开源轻量 Agent，适合本地部署
-Jupyter AI 增强版：代码与文档协同，适合科研与数据分析
```

## 三 总结

```
1.编程助手：Cursor（复杂全栈）、Copilot（日常团队）、Claude Code（企业高可靠）
2.Agent 开发：LangGraph Studio（复杂可控）、Flowise（无代码快速）、Dify（企业落地）
3.IDE 选型：Cursor 全能，Continue 轻量，OpenDevin 自主，Manus 企业级
```
