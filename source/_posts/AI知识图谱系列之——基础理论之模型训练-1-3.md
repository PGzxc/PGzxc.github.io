## 二 AI理论基础/模型训练

### 2.1 什么是预训练(Pretraining)？

```
1.概念
预训练是在 大规模数据上训练通用模型。

2.数据来源：
 -互联网文本
 -代码
 -书籍
 -论文

3.目标：
学习通用知识

4.例如：
 -GPT
 -BERT
 -Llama
```

### 2.2 什么是微调(Fine-tuning)？

```
1.概念
Fine-tuning 是在 预训练模型基础上继续训练特定任务数据。

2.优点：
 -成本低
 -效果好
 -训练时间短

3.应用：
 -医疗模型
 -法律模型
 -客服模型
```

### 2.3 什么是Instruction Tuning？

```
1.概念
Instruction Tuning 是通过 指令数据训练模型遵循指令能力。

2.数据格式：
 -Instruction
 -Input
 -Output

3.例如：
User：解释机器学习
Assistant：……
```

### 2.4 什么是SFT？

```
1.概念
SFT（Supervised Fine-Tuning）是使用 人工标注数据进行监督微调。

2.作用：
让模型学会对话和回答问题

3.备注
SFT通常是 LLM训练的第二阶段。
```

### 2.5 RLHF是什么？

```
1.概念
RLHF(Reinforcement Learning from Human Feedback)是通过 人类反馈优化模型行为 的方法。

2.流程：

Pretraining
↓
SFT
↓
Reward Model
↓
强化学习优化

3.目标：
让回答更符合人类偏好
```

### 2.6 RLAIF是什么？

```
1.概念
RLAIF(Reinforcement Learning from AI Feedback)是使用 AI生成反馈替代人工标注。

2.优势：
 -降低成本
 -扩大数据规模
```

### 2.7 DPO为什么成为主流？

```
1.概念
DPO(Direct Preference Optimization)是一种 直接优化偏好数据的训练方法。

2.优点：
 -不需要奖励模型
 -不需要强化学习
 -训练更稳定
 -实现更简单

因此逐渐替代 RLHF。
```

### 2.8 什么是PEFT？

```
1.概念
PEFT(Parameter Efficient Fine-Tuning)是一种 只微调少量参数的技术。

2.优点：
 -降低训练成本
 -减少GPU需求
```

### 2.9 LoRA的原理是什么？

```
1.概念
LoRA（Low-Rank Adaptation）通过 低秩矩阵分解更新模型权重。

2.公式：
W + ΔW
ΔW = A × B

3.优点：
-只训练少量参数
-显存需求低
```

### 2.10 QLoRA是什么？

```
1.概念
QLoRA = 4bit量化 + LoRA微调

2.优势：
 -显存占用极低
 -单GPU即可训练大模型
```
