## 二 100问

### 2.1、基础概念（1～10）

```
1. 什么是AI？
用算法让机器具备感知、学习和决策能力

2. 什么是机器学习？
从数据中学习规律，而不是写死规则

3. 什么是深度学习？
使用多层神经网络进行学习

4. 什么是LLM？
大规模训练的语言模型（如 GPT-5.4）

5. LLM的核心能力？
理解、推理、生成

6. 什么是Transformer？
基于注意力机制的模型架构

7. 什么是Attention？
模型关注重要信息的机制

8. 什么是Token？
模型处理的最小单位（字/词/子词）

9. 什么是上下文窗口？
模型一次能处理的最大Token数

10. 什么是多模态？
同时处理文本、图片、语音等
```

### 2.2、Prompt工程（11～20）

```
11. 什么是Prompt？

12. 什么是Prompt Engineering？

13. Zero-shot / Few-shot区别？
是否提供示例

14. System Prompt作用？
定义AI行为规则

15. Prompt优化方法？
明确指令 + 提供示例 + 限制输出

16. 什么是Chain-of-Thought？
引导模型一步步思考

17. 什么是Self-Consistency？
多次生成取最优答案

18. Prompt Injection是什么？
恶意输入影响模型行为

19. 如何防御Prompt攻击？
输入过滤 + 权限隔离

20. Prompt的本质？
一种“弱编程语言”
```
