## 二 学习内容

### 2.1 应用模型

```
-UIAbility 生命周期
-Page 页面与路由跳转
-AbilityStage 管理
```

### 2.2 数据管理

```
-本地存储（Preferences、数据库 RDB、文件系统）
-网络请求（HttpRequest、WebSocket）
-分布式数据管理（跨设备数据同步）
```

### 2.3 系统能力调用

```
-媒体（相机、相册、音视频播放）
-传感器（加速度计、陀螺仪、定位）
-通知、任务调度
```

### 2.4 工程实践

```
-工程目录结构
-配置文件（module.json5、app.json5）
-调试与日志（hilog）
```
