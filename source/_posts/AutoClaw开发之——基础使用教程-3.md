## 二 基础教程

### 2.1 简单用法(对话即执行)

```
1.在 AutoClaw 输入：帮我总结最近AI趋势

2.它会：
 -搜索资料
 -分析总结
 -输出结构化内容
```

### 2.2 指令写法

```
1.推荐结构：
任务 + 约束 + 输出格式

2.示例：

帮我写一篇关于RAG的博客
要求：
1. 面向开发者
2. 包含架构图
3. 使用Markdown格式
```
### 2.3 多步骤任务(Agent能力)

```
一、示例：
帮我：
1. 搜索AI面试题
2. 整理分类
3. 输出PDF

二、AutoClaw会自动：

-拆解任务
-执行链路
-组合结果
```
