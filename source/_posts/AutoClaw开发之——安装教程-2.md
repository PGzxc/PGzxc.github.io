## 二 下载及安装

### 2.1 下载

```
1.进入官网：https://autoglm.zhipuai.cn/autoclaw
2.下载AutoClaw
3.支持：Windows/macOS
```

### 2.2 安装

```
一、步骤：
-下载安装包
-双击安装
-打开应用
-登录账号

二、无需：Node.js/Python/命令行

三、做到：下载 → 打开 → 用
```
## 三 初始化配置

```
首次启动：
-登录智谱账号
-选择模型（默认即可）
-开启浏览器控制权限（重要）
```
