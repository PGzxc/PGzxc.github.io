## 一 概述

在虚函数中使用成员名限定可以强行解除动态联编  

<!--more-->

## 二 示例演示及输出

### 2.1 代码

```
#include<iostream>
using namespace std;
class A
{
	public:virtual int get(){return 0;}
};
class B:public A
{
	public:int get(){return 1;}
};
int main()
{
	B b;
	A *p=&b;
	cout<<p->get()<<endl;
	cout<<p->A::get()<<endl;
	return 0;
}
```

### 2.2 输出结果

```
1
0
```

### 2.3 代码说明

* 使用成员限定p->A::get()可以指定输出
