## 二 关系
### 2.1 关系说明
```
a[i][j]=a[i-1][j-1]+a[i-1][j];
```

### 2.2 举例说明

```
a[2][1]=a[1][0]+a[1][1];
```

## 三 示例代码及结果输出

### 3.1 代码

```
#include<iostream>
using namespace std;
int main()
{
	const int n=10;
	int a[n][n];
	for(int i=0;i<n;i++)
	{
		for(int j=0;j<=i;j++)
		{
			if(j==0||j==i)
			{
				a[i][j]=1;
			}else{
				a[i][j]=a[i-1][j-1]+a[i-1][j];
			}
			cout<<a[i][j]<<"\t";
		}
		cout<<endl;
	}
	return 0;
}
```
