## 二 代码及说明

### 2.1 代码

```
#include<iostream>
using namespace std;

int main() {
	int a = 2, b;
	b = 6 + a;
	a+=6;
	cout<<a<<endl;
	return 0;
}
```

### 2.2 说明

* a+=6;这一行语句将赋值符"="与数学运算符"+"联合起来使用，它表示将右值与左值相加的结果赋给左值。它的作用与a=a+6是相同的。
* 注意："+="读作"加等"
* 由于a的值为2，与6相加后等于8，因此运算结束后a的值为8

### 三 运算符总结

* 减等(-=)
* 乘等(*=)
