## 一 概述

逻辑"或"运算符"||"稍微比逻辑"与"运算符"&&"要同情道理一些，它只需要逻辑表达式一边的条件为真，那么整个表达式为真。  

<!--more-->

## 二 代码及结果

### 2.1 代码

```
#include<iostream>
using namespace std;

int main() {

	cout << "请输入一个大于1的整数：\n";
	int x;
	cin >> x;
	if (x > 1 || x < 100) {
		cout << "x大于1小于100\n";
	}
	return 0;

}
```

### 2.2 结果

```
请输入一个大于1的整数：
0
x大于1小于100
```
