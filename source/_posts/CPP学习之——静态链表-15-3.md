## 一 概述

本节两则程序中所有结点的数据在内存中的分布都是在编译时就确定好了的，不会在运行时再进行动态地分配空间，比如说我们不能在程序运行时动态地对各个结点进行创建、添加和删除的工作，这都是在运行前都已经确立好了的，我们只能在编译前修改指向结点的指针，这种链表叫"静态链表"  
<!--more-->

## 二 示例演示及结果输出

### 2.1 代码

```
#include<iostream>
using namespace std;
class book
{
	public:
		int num;
		float price;
		book *next;
};
int main()
{
	book x,y,z,*head,*p;
	x.num=10000;
	x.price=14.5;
	y.num=20000;
	y.price=23.4;
	z.num=30000;
	z.price=45.6;
	head=&x;
	x.next=&y;
	y.next=&z;
	z.next=NULL;
	p=head;
	while(p!=NULL)
	{
		cout<<p->num<<","<<p->price<<endl;
		p=p->next;
	}
	return 0;
}
```

### 2.2 输出结果

```
10000,14.5
20000,23.4
30000,45.6
```
