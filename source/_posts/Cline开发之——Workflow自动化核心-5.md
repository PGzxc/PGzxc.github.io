## 二 Workflow工作流

### 2.1 创建工作流(官方方式)

```
1.在项目中创建：
.clinerules/workflows/

2.然后输入：
/create-new-workflow.md

3.Cline 会引导你：
 -输入目标
 -拆分步骤
 -自动生成流程
```

### 2.2 工作流示例

```
1.示例：
开发接口 → 写代码 → 运行测试 → 提交代码

2.后续
一次定义，反复执行
```

### 2.3 官方最佳实践

```
-从小流程开始
-模块化拆分
-写清注释
-放入 Git 管理
```
