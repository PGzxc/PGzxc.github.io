## 二 什么是 Deep-Live-Cam

### 2.1 项目地址

```
https://github.com/hacksider/Deep-Live-Cam
```

### 2.2 什么是 Deep-Live-Cam

```
Deep-Live-Cam 是一个实时换脸(Face Swap)开源项目，支持：
 -图片换脸
 -视频换脸
 -摄像头实时换脸（直播级别）
```

### 2.3 核心特点

```
单张人脸即可驱动换脸
支持 GPU 加速（CUDA / CoreML / DirectML）
实时处理（低延迟）
GUI + CLI 双模式
```

### 2.4 本质

```
一个“实时 DeepFake 引擎"
```

## 三 能做什么？

```
典型应用场景：

 -短视频创作（抖音/快手）
 -虚拟主播 / VTuber
 -表情包生成
 -游戏角色替换
 -直播变脸
```

## 四 技术亮点

ONNX Runtime 推理

一个典型 AI 多组件融合工程
