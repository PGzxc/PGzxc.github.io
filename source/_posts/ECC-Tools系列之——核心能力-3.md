## 二 核心能力

### 2.1 Skills(技能系统)

```
1.作用：

把“经验”变成“可复用能力”

2.示例：
- /tdd-workflow → 强制测试驱动开发
- /security-review → 自动安全检查
- /plan → 自动拆解任务
```

### 2.2 Agents(子代理)

```
1.作用：
专项 AI 工具

2.典型：
- code-reviewer（代码审查）
- planner（任务规划）
- security auditor（安全分析）
```

### 2.3 Hooks(自动化)

```
1.作用：
自动执行规则

2.例如：
- 提交前格式检查
- PR 自动扫描
- 安全校验
```

### 2.4 Continuous Learning(持续学习)

```
1.机制：
- 观察行为
- 提炼模式
- 生成“instincts”

2.AI 越用越懂你 
```
