## 一 概述

```
本文介绍：
 - 在搜索框中检索内容
 - 导出(保存)当前文件列表
```

<!--more-->

## 二 在搜索框中检索内容

### 2.1 在搜索框中指定目录

```
C:  //检索C盘下的内容
```

### 2.2 使用过滤器只看某种类型

```
C: ext:txt //检索C盘下的txt文件
```

## 三 导出(保存)当前文件列表

### 3.1 导出为CSV文件

```
1、操作
文件——>导出——>CVS文件(*.csv)

2、问题
— 使用Excel打开可能出现乱码
-乱码问题：Notepad++ / VS Code 转换
```

### 3.2 导出为txt文件

```
1、操作
文件——>导出——>文本文件(*.txt)

2、说明
- 不存在乱码问题
```

### 3.3 导出为EFU文件

```
1、操作
文件——>导出——>EFU Everything文件列表(*.efu)

2、说明
- 使用Everything软件打开
```

## 四 总结对比

|       目的       |      方法      |        输出文件        | 是否离线可查看 |
| :--------------: | :------------: | :--------------------: | :------------: |
|   导出搜索结果   | 导出为 CSV/TXT |     .csv / .txt      |  否（纯文本）  |
|   备份整个索引   |   导出数据库   |          .db           |       否       |
| 保存目录文件清单 | 导出为.efu |          .efu          |  ✅ 可离线查看  |
|  命令行生成列表  |     es.exe     | .txt / .csv / .efu |     ✅ 可选     |
