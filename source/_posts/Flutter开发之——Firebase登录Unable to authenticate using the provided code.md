## 三 通过代理执行过程

### 3.1 添加代理

```
C:\Windows\System32>set HTTP_PROXY=http://127.0.0.1:7890
C:\Windows\System32>set HTTPS_PROXY=http://127.0.0.1:7890
```

说明：

* 在cmd终端中执行上诉操作
* `http://127.0.0.1:7890`为Clash对应端口，其他根据软件切换即可
