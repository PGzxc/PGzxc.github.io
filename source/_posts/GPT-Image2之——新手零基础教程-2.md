## 三 核心能力演示

### 3.1 文本生成图片

1-输入

```
一个80年代中国农村，夏天收割麦子的场景，阳光强烈，写实风格
```

2-输出-高质量写实图片(可用于设计/内容生成)

![][1]



### 3.2 图像编辑(局部修改)

```
1.支持：
-替换背景
-修改人物
-增加元素
-LOGO 抠图

2.典型场景：
比如：之前做的「椭圆 logo 抠图」就属于这一类
```

```
1-输入：
将这张照片转换为宫崎骏动画风格

2-输出：
动漫风 / 插画风 / 油画风等
```
