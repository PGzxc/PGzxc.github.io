## 二 创建与用户名同名的仓库

```
1、登录你的 GitHub 账号
2、点击右上角 ➕ → New repository

3、填写：
-Repository name：必须与你的 GitHub 用户名完全一致（大小写敏感）
-Description（可选）：比如“个人简介”
-勾选 "Add a README file"

4、创建完成后，你会看到一个默认的 README.md 文件，自动展示在你的 GitHub 首页上
```
