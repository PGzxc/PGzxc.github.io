## 二 通过Eclipse开发HTML

* Eclipse中依次点击：File——>New——>Static Web Project

  ![][1]
  
* 在弹出的窗口中输入要创建WebPage的名称，点击Finish

  ![][2]
  
* 在webContent上右键，依次选择：New——>HTML File

  ![][3]
  
* 输入html页面的名称，点击Finish

  ![][4]

* 在页面中输入内容如`<h1>网页内容</h1>`然后右键，选择打开方式

  ![][5]
