## 一 HandBrake 简介

```
HandBrake 是一款 免费、开源的多平台视频转码工具，
可将几乎任何视频格式转换为主流兼容的现代格式（如 MP4、MKV、WebM）。
官网地址：https://handbrake.fr
```

<!--more-->

## 二 核心特点

|      功能      |                             说明                             |
| :------------: | :----------------------------------------------------------: |
| 支持多种输入源 |            几乎所有视频文件、DVD、蓝光(未加密)等             |
|    输出格式    |                        MP4、MKV、WebM                        |
|   编码器支持   |              H.264、H.265（HEVC）、AV1、VP9 等               |
|    设备预设    |         内置手机、平板、电视、Web 等多种设备预设参数         |
|    批量转码    |              支持队列处理，可一次性转码多个文件              |
|    高级设置    | 可调整比特率、画质、帧率、滤镜（降噪、去交错、裁剪、缩放）、字幕及音轨 |
|   字幕与章节   |             支持内嵌或外挂字幕、章节标记导入导出             |

## 三 限制与注意事项

```
-不支持解密或绕过 DRM(数字版权保护)的内容。
-不具备视频剪辑或合并功能，仅限转码与压缩。
-输出容器有限（仅支持 MP4、MKV、WebM）。
```

## 四 支持平台

|  系统   |              状态              |
| :-----: | :----------------------------: |
| Windows |         官方安装包支持         |
|  macOS  |          提供原生版本          |
|  Linux  | 提供 CLI(命令行)与图形界面版本 |

## 五 常见应用场景

```
-将旧视频文件转码为手机、平板可播放格式。
-压缩视频以减少文件体积（如从 4GB 压缩到 1GB）。
-为 NAS 或家庭影院整理兼容视频格式（H.264/H.265）。
-使用 CLI 版本进行 自动化批量转码（适合服务器或 NAS 部署）。
```

## 六 实用技巧

```
-若追求「高画质+较小体积」，推荐使用 恒定质量（Constant Quality）模式。
-若输出给旧设备或网页播放，推荐容器：MP4 + H.264 + AAC。
-可利用「设备预设」快速选择输出参数（如 Apple、Android、Roku、YouTube 等）。
-若需字幕支持，建议选择 MKV 容器，兼容性更好。
```

## 七 为什么很多人喜欢 HandBrake

```
-完全 免费 + 无广告 + 开源。
-界面直观、操作简单，新手也能轻松使用。
-支持命令行和批量任务，适合开发者与影音工作流自动化。
-跨平台一致体验，适合在 PC、Mac、Linux、NAS 上统一使用
```
