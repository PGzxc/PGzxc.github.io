## 一 概述

```
群晖提供HTTP本地服务的是Web Station
- 将视频文件放到Web目录/web/Video目录选
- 根据/web/Video视频生成iptv文件
- 局域网访问iptv并播放
```

<!--more-->

## 二 Web Station安装

### 2.1 安装方式

```
1、官方套件：套件中心——>所有套件——>搜索Web Station并安装
2、手动安装
 - 下载中心：https://www.synology.cn/zh-cn/support/download/DS1821+?version=7.2#packages
 - DMS套件：下载Web Station
 - 套件中心：手动安装
```
