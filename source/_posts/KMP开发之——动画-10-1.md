## 二 Compose 动画的核心思想

### 2.1 动画 ≠ 定时器

```
1-传统思维：
Timer/Handler/invalidate

2-Compose 思维：
State → Animation → UI

3-说明：
动画只是 State 的一种“过渡形式”
```

### 2.2 为什么 Compose 动画天然跨端？

```
动画运行在 Compose Runtime
不依赖平台 View / Layer
Android / iOS / Desktop 统一模型
```

## 三 动画

### 3.1 最基础动画：animate*AsState

1-animateFloatAsState

```
@Composable
fun ScaleBox() {

    var expanded by remember { mutableStateOf(false) }

    val scale by animateFloatAsState(
        targetValue = if (expanded) 1.2f else 1f,
        animationSpec = tween(300)
    )

    Box(
        modifier = Modifier
            .size(100.dp)
            .graphicsLayer {
                scaleX = scale
                scaleY = scale
            }
            .clickable { expanded = !expanded }
            .background(Color.Blue)
    )
}

说明：最常用、最安全
```

2-常见 animate*AsState 家族

|         API          | 说明 |
| :------------------: | :--: |
| animateFloatAsState  | 数值 |
|   animateDpAsState   | 尺寸 |
| animateColorAsState  | 颜色 |
| animateOffsetAsState | 位移 |

### 3.2 动画参数 animationSpec

1-tween(补间动画)

```
1-代码
tween(
    durationMillis = 400,
    easing = FastOutSlowInEasing
)

2-常用 easing：
-LinearEasing
-FastOutSlowInEasing
-LinearOutSlowInEasing
```

2-spring(物理动画)

```
1-代码
spring(
    dampingRatio = Spring.DampingRatioMediumBouncy,
    stiffness = Spring.StiffnessLow
)

2-适合：
-拖拽释放
-回弹
-弹性反馈
```

### 3.3 组合动画：多个属性一起动

```
1-代码

val size by animateDpAsState(
    targetValue = if (expanded) 120.dp else 80.dp
)

val color by animateColorAsState(
    targetValue = if (expanded) Color.Red else Color.Gray
)

2-说明
共享同一个 State 即可
```

### 3.4 Visibility 动画(高频)

1-AnimatedVisibility

```
1-代码
AnimatedVisibility(
    visible = expanded,
    enter = fadeIn() + expandVertically(),
    exit = fadeOut() + shrinkVertically()
) {
    Text("Hello KMP")
}

2-适用于：
-弹出区域
-条件 UI
-展开 / 收起
```

2-AnimatedContent(内容切换)

```
1-代码
AnimatedContent(
    targetState = expanded
) { target ->
    if (target) {
        Text("Expanded")
    } else {
        Text("Collapsed")
    }
}

2-说明
自动处理内容切换动画
```

### 3.5 Infinite 动画(循环)

1-rememberInfiniteTransition

```
val infinite = rememberInfiniteTransition()

val alpha by infinite.animateFloat(
    initialValue = 0.3f,
    targetValue = 1f,
    animationSpec = infiniteRepeatable(
        animation = tween(800),
        repeatMode = RepeatMode.Reverse
    )
)
```

2-常用于

```
Loading
呼吸动画
闪烁提示
```

### 3.6 动画与 Modifier

1-graphicsLayer

```
1-代码
raphicsLayer {
    translationX = offsetX
    rotationZ = rotation
}
2-说明：
性能最好，走底层变换
```

2-Modifier.offset vs graphicsLayer

|     方式      |    特点    |
| :-----------: | :--------: |
|    offset     |  参与布局  |
| graphicsLayer | 不触发布局 |

## 四 KMP 动画最佳实践

```
1.动画驱动 State
2.缩小重组范围
3.动画只影响必要 UI
4.多端统一动画参数
```

## 五 小结

```
动画本质是 State 的过渡
animate*AsState 是基础核心
tween / spring 覆盖 80% 场景
AnimatedVisibility / Content 非常高频
KMP 中动画是真正的“跨端一致体验”
```
