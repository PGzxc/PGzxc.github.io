## 一 概述

```
本文介绍：
出厂设置 = 擦除所有数据 + 删除所有账号设置 + 重装 macOS
最终效果：像刚买来的全新机器
机型：苹果 M 系列与 Intel 机型
```

<!--more-->

## 二  苹果M系列(M1/M2/M3/M4)

### 2.1 适用机型

```
适用于 2020 年后的 MacBook / iMac / Mac mini / Mac Studio。
```

### 2.2 步骤

```
一、进入恢复模式
关机 → 按住 电源键不放 → 出现“正在载入启动选项”
选择：【选项 Options】 → 继续 Continue

二、选择管理员账户 → 输入密码
三、打开磁盘工具，抹掉系统盘
-磁盘工具（Disk Utility） → 左上角选择“显示所有设备”
-选择系统盘（Macintosh HD）→ 点击 抹掉 Erase
名称：Macintosh HD
格式：APFS
分区方案：GUID（GUID Partition Map）

四、回主界面 → 重新安装 macOS
-选择：【重新安装 macOS Reinstall macOS】
-按提示安装系统即可。
-完成后会出现“欢迎”界面，表示已恢复出厂状态。
```

## 三 Intel 处理器 Mac(2019 年及以前)

### 3.1 说明

```
适用版本：macOS Catalina 及更早版本
```

### 3.2 步骤

```
一、重启 → 按住 Command + R 进入恢复模式。
二、磁盘工具 → 抹掉系统盘：格式选择：Mac OS 扩展（日志式）
三、回到主界面 → 重新安装 macOS：按提示即可完成系统重装与出厂重置。
```

## 四 最快捷的一键出厂设置(Monterey 及以上)

### 4.1 要求

```
如果你的 Mac 是 Apple Silicon 或带 T2 芯片的 Intel，并且系统版本 ≥ macOS Monterey：
你可以像 iPhone 一样一键恢复出厂设置：
```

### 4.2 步骤

```
路径：系统设置 → 通用 → 传输或还原 → 抹掉所有内容和设置
输入密码 → 自动清空所有数据 → 自动退出 Apple ID → 自动重新安装干净系统。
```

### 4.3 说明

```
这是最简单、最推荐的方法
```

## 五 注意事项及总结

### 5.1 注意事项

```
恢复出厂设置会 永久删除所有数据（不可恢复）
请务必提前备份：Time Machine或者手动拷贝重要资料
重装系统必须连接网络（建议 Wi-Fi 或网线）
```

### 5.2 总结

```
2021 年后的 Mac（M1/M2/M3/M4）：
长按电源键 → 选项 → 磁盘工具抹掉 Macintosh HD → 重新安装 macOS = 出厂设置 + 重装系统。
```
