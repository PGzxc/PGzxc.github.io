## 一 概述

```
NAS未绑定Mac和IP前，每次启动，NAS的IP地址可能发生变化。
TV等终端设备访问时，IP发生变化，导致访问地址改动。
```

<!--more-->

## 三 IP与Mac相关操作(路由器终端)

### 3.1 绑定操作

3-查看ip和Mac地址，通过指令绑定

```
arp -s 192.168.1.7 00:00:C0:39:32:F8
```

![][5]

### 3.2 解绑操作(解除IP与Mac绑定)

```
arp -d 192.168.1.7
```

![][6]
