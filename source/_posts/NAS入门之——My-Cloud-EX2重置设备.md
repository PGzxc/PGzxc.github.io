## 一 概述

* 因何操作无法访问NAS
* 重置设备重新设置

<!--more-->

## 二 因何操作无法访问NAS

### 2.1 原因

```
网络服务 DHCP设置了Google DNS
```

### 2.2 步骤(不要进行此操作)

1、登录后，点击：设置—>网络—>网络服务—>IPV4网络—>DHCP

![][1]

2、点击DHCP后，点击Google Public DNS并应用

![][2]

### 2.3 结果

```
重启后，无法查找到NAS设备
```

## 三 重置设备重新设置

### 3.1 重置前说明

```
1-硬盘数据不会丢失
2-登录密码会丢失，跳转到登录页面后，用户名admin，密码不需要设置点击后修改
3-网络模式恢复默认值
```

### 3.2 找到设备的重置按钮

根据自己的设备找到重置按钮，如下图，长按5秒左右

![][3]

### 3.3 登录后设置新密码

```
登录后，用户名admin，不需要设置密码，点击登录，需要设置新密码
```
