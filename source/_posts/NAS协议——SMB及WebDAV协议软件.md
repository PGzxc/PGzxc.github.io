## 三 如何选择

### 3.1 完全免费且好用的首选(跨平台推荐)

```
1. Android：X-plore（带广告可忍）或 Amaze（纯开源）
2. iOS：Documents by Readdle（日常够用）
3. Windows：Cyberduck / WinSCP
4. macOS：Cyberduck
```

### 3.2 性价比最高(推荐买断)

```
1.Android：Solid Explorer（一次性13元左右，体验极佳）
2.iOS：GoodReader 或 FE File Explorer Pro
3.Windows/macOS：Mountain Duck(挂载成本地盘最方便，298元终身)
```
