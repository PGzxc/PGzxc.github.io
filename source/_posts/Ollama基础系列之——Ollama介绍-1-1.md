## 二 什么是 Ollama？

### 2.1 概念

```
Ollama 是一个用于在本地运行大语言模型(LLM)的开源工具，
可以让你在电脑上直接运行类似 GPT 的模型，而无需依赖云服务。
```

### 2.2 核心特点

```
1.本地运行(无需联网)
2.数据隐私安全（不上传服务器）
3.开源免费
4.支持多模型(Llama、Qwen、DeepSeek等)
```

## 三 Ollama 解决了什么问题？

```
1、传统 AI（如 ChatGPT）的问题：
 -依赖云服务
 -有 Token 成本
 -数据隐私风险
 
2、Ollama 的优势：
-完全离线运行
-无调用费用
-自由控制模型

3、本质：
本地 AI 运行时（类似 Docker 之于容器）
```

## 四 Ollama 能做什么？

```
1.AI 对话(Chat)
2.代码生成(Copilot 替代)
3.文本总结 / 翻译
4.向量 Embedding
5.本地知识库（RAG）
6.多模态（图像模型）
```
