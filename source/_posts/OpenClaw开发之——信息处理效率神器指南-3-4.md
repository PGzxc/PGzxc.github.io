## 一 概述

```
本篇介绍三大信息处理类效率神器，
帮助你实现网页数据采集、长文档智能总结以及实时联网搜索能力，大幅提升信息获取与处理效率。
```

<!--more-->

## 二 安装命令

```
# 一键安装信息处理三大神器
clawhub install web-scraper document-summarizer web-search
```

## 三 技能详解

### 3.1 web-scraper — 网页抓取与监控专家

```
-定时抓取指定网页数据
-支持价格监控、文章更新监控、招聘信息采集
-可将抓取结果结构化保存
```

### 3.2 document-summarizer —长文档总结大师

```
-支持 PDF、Word、网页长文、论文、书籍章节
-提取核心要点、生成思维导图式总结
-支持中英文翻译与多版本总结
```

### 3.3 web-search —实时联网搜索增强

```
-突破本地模型知识截止日期限制
-支持实时搜索最新资讯、论文、技术文档
-可与本地模型混合使用，提供最新准确信息
```
