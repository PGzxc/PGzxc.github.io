## 二 OpenCode相关概念

### 2.1 1句话理解

```
OpenCode = 面向开发者的 AI 编程 CLI + Agent 工具
```

### 2.2 类似

```
Claude Code
Codex CLI
Continue CLI
```

### 2.3 特点

```
-开源
-可接入任意模型
-支持本地模型（Ollama）
-可自动执行任务（Agent能力）
```

## 三 核心能力

|    功能    |          说明           |
| :--------: | :---------------------: |
|  代码生成  | 写代码 / 补全 / 改 Bug  |
| Agent执行  | 自动分析项目并执行任务  |
| 多模型支持 | GPT / Claude / 本地模型 |
|  CLI操作   |     直接在终端使用      |

## 四 适合人群

后端 / 前端开发
AI工程师
