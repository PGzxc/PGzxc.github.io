## 二 createColorPicker语法介绍

### 2.1 语法

```
createColorPicker([value])
```

### 2.2 说明

* 创建一个颜色拾取器
* 可以级联使用p5.Element的语法

### 2.3 参数及返回值

#### 参数

value：元素的默认颜色

#### 返回值

p5.Element

## 三 示例


### 3.1 代码

```
let colorPicker;

function setup() {

  createCanvas(100, 100);
  colorPicker = createColorPicker("#ed225d");
  colorPicker.position(0, height + 5);

}

function draw() {
  background(colorPicker.color());
}
```
