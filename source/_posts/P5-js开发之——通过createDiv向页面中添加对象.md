## 二 createDiv语法介绍

### 2.1 语法

```
createDiv([html])
```

### 2.2 参数及返回值

|         参数          |   返回值   |
| :-------------------: | :--------: |
| 创建div时默认的字符串 | p5.Element |

## 三 示例

### 3.1 示例一

#### 代码

```
function setup() {
  let div = createDiv('createDiv');
  div.style('font-size', '16px');
  div.position(10, 0)
}
```

### 3.2 示例2
#### 代码

```
function setup() {
  let div2 = createDiv("DIV中内容")
            .style('font-size', '32px')
            .style('color', 'red')
            .style('top', '200px')
            .mousePressed(() => {
                console.log("mousePress");
            })
            .center()
          .html("文章内容", true);
}
```
