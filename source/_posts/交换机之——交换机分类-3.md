## 二 按管理能力分类(最常用的划分方式)

### 2.1 非管理型交换机(最常见)

```
1、说明：即插即用，无需配置
2、适合：家庭、宿舍、小办公室
3、特点：
-插上就能用
-价格便宜
-无 VLAN、QoS、端口绑定等高级功能
-稳定、简单、不易出错

家用 NAS / 电脑 / 电视 / 游戏机 → 99% 用非管理型即可
```

### 2.2 管理型交换机(可配置)

```
1、适合：企业、弱电工程、工作室、监控、校园网

2、支持高级功能，例如：
-VLAN 网络隔离
-QoS（流量优先级）
-端口聚合（LACP）
-ACL（访问控制）
-端口监控 / 镜像
-三层路由（部分型号）

分为两种：

2.1 智能管理型（Simple / Smart）
-比非管理多一些功能
-配置简单
-性价比高
-常用于办公场景

2.2 全管理型（Fully Managed）
-高级配置非常丰富
-企业机房常用
-可做复杂的网络拓扑
```

## 三 按速率分类(影响 LAN 速度)

### 3.1 百兆(100M)

```
已过时，不推荐
网络设备普遍进入千兆时代
```

### 3.2 千兆(1G，家用主流)

```
最常见
适合家庭和大多数办公室
NAS / 电脑之间可达 110MB/s
```

### 3.3 2.5G(NAS/WiFi7 常用)

```
-家用高速新趋势
-WiFi 7、NAS高速备份需要
-比 10G 更便宜、发热更低

强烈推荐：NAS 用户、WiFi 7 用户
```

### 3.4 10G(企业/发烧友)

```
1、有两种接口：
-10G SFP+（光口）
-10G RJ45（电口）

2、适合：
-高性能 NAS
-影棚/视频工作室大文件传输
-企业机房
```

### 3.5 混合多速率(常见)

```
例如：
-8 口千兆 + 2 口 10G
-6 口 2.5G + 2 口 10G

适合中高端 NAS 组网。
```

## 四 按供电方式分类

### 4.1 普通交换机(非 PoE)

```
只传输网络
家用最常见
```

### 4.2 PoE 交换机(Power over Ethernet)

```
1、说明：一根网线＝网络 + 电源
2、适合：摄像头（IPC）、WiFi AP（如 TP-LINK、Ubiquiti）、门禁 / 对讲机
3、PoE 规格常见：PoE（48V）、PoE+、PoE++（大功率）
```

## 五 按层级分类(网络协议模型)

### 5.1 二层交换机(L2 Switch)

```
最常见
使用 MAC 地址转发
大部分家用交换机属 L2
```

### 5.2 三层交换机(L3 Switch)

```
具备路由功能
可实现 VLAN 间通信
实际上介于“路由器”和“交换机”之间

适合中大型企业。
```

### 5.3 四层及以上(L4、L7)

```
用于数据中心
支持更复杂的策略
常用于运维、大型服务器集群

家用基本不会碰到
```

## 六 按安装方式分类

### 6.1 桌面式交换机

```
适合家用、桌面摆放
静音，无风扇
```

### 6.2 机架式交换机(1U / 2U)

```
企业弱电机房使用
支持上机柜
端口众多（24/48 口）
容量大、可靠性高
```

## 七 按端口类型分类

### 7.1 全电口(RJ4)

```
最常见
直接插网线
```

### 7.2 光电混合(RJ45 + SFP/SFP)

```
1、例如：
-千兆电口 + 千兆光口
-千兆电口 + 10G SFP+ 光口

2、适合：
-多楼层网络布线
-NAS / 服务器高速互联
```

## 八 按设备用途分类

### 8.1 家用交换机

```
-千兆为主
-少量 2.5G（NAS 用户）
-即插即用
```

### 8.2 办公交换机

```
管理型
有 VLAN/QoS
```

### 8.3 工业交换机

```
防雷、防潮、高温
户外摄像头、公路、工地
```

### 8.4 监控交换机

```
PoE 功能
专为摄像头使用
有大功率供电
```
