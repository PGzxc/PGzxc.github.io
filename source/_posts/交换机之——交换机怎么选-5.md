## 一 概述

```
本文介绍：
 - 家用/办公都能照搬的交换机选购指南
 - 千兆 vs 2.5G vs 10G
```

<!--more-->

## 二 先看最关键的决定因素

### 2.1 宽带带宽

```
100M/200M/500M/1000M 宽带：千兆交换机足够
2000M（2G）宽带：最低 2.5G 交换机
3000M 以上：建议部分口上 10G
```

### 2.2 局域网需求(NAS、电脑互传、视频剪辑、Plex/Jellyfin、监控等)

|                  需求场景                   |       推荐档位        |
| :-----------------------------------------: | :-------------------: |
|       普通家庭、上网、电视、智能家居        |         千兆          |
|         有 NAS，想提高文件拷贝速度          | 2.5G 交换机（最香档） |
| 大文件编辑、视频剪辑（4K/8K）、多进程高吞吐 |      10G 交换机       |
|              发烧友、家庭机房               |   混合：10G + 2.5G    |

### 2.3 布线情况

```
室内网线是 Cat5e/6 → 最高能稳定跑到 2.5G
Cat6 正规布线 → 稳定 10G
Cat6A、Cat7/8 → 未来可升级 10G～25G
```

## 三 千兆 vs 2.5G vs 10G：核心对比表

|    特性    | 千兆（1G）  |     2.5G     |           10G            |
| :--------: | :---------: | :----------: | :----------------------: |
|  实际速度  |   110MB/s   |   280MB/s    |         1000MB/s         |
|  网线要求  | Cat5e 即可  |  Cat5e 即可  | Cat6（短距离）推荐 Cat6A |
|    发热    |     低      |      中      |            高            |
|    噪音    | 100% 无风扇 | 大部分无风扇 |      高速口常带风扇      |
|    价格    |    便宜     |  性价比第一  |           昂贵           |
| 家庭推荐度 |    5颗星    |    5颗星     |          3颗星           |

## 四 买交换机最推荐的分档

### 4.1 档位 A：预算有限 / 普通家庭

```
1、档位：选 千兆交换机（1G）
2、适合：
-只是上网、看电视、智能家居
-没有 NAS 或本地大文件需求

3、推荐品牌：绿联、TP-Link、迅捷、华为、H3C（家用款）

4、购买建议：
一定买金属外壳散热更好
尽量买 8 口，不要买 5 口（容易不够用）
```

### 4.2 档位 B: 99% 家庭最推荐:2.5G 交换机(性价比最高)

```
1、档位
-最香！
-NAS 用户首选！

2、适合：
-有 NAS（群晖 / 威联通 / 小型自建）
-想提高电脑 ↔ NAS 拷贝速度
-想让家庭网络提升一个档次
-想未来升级 2G/3G 宽带

3、优势：
-使用同样的 Cat5e 网线即可跑满速
-不贵：8 口无管理 2.5G ≈ 400–600 元
-发热低，基本无噪音

4、经典需求案例：
-Windows/Mac ↔ 群晖 传文件更快
-电影剪辑 NAS 上直接编辑（Lightroom/Premiere）

5、推荐配置：

10G(上行到NAS/主路由，可选)
+ 
全屋 2.5G（强烈推荐）
```

### 4.3 档位 C：专业/视频剪辑/高性能 NAS：10G 交换机

```
1、适合：
-4K/8K 视频编辑
-大文件经常在本地局域网传输
-高性能 NAS（如群晖 DS923+/DS1522+）
-家用机房/实验室/开发环境

2、注意：
-10G 发热大、功耗高、多为有风扇
-需要 Cat6 / Cat6A 以上网线
-价格贵（8 口 10G 动不动 1500–3000 元）

3、推荐品牌：
-QNAP（威联通）
-MikroTik
-梦想机（Dream Machine Pro 配 SFP+）
-H3C/华三、万兆企业级交换机
```

## 五 家庭用网推荐方案

### 5.1 方案 1：普通家庭(无 NAS)

```
1、方案
光猫 → 路由器 → 千兆交换机 → 房间网口

2、预算：150～200 元
3、性能：足够使用。
```

### 5.2 方案 2：家里有 NAS(推荐）

```
1、方案
主路由(1G/2.5G) → 2.5G 交换机 → NAS / 电脑 / 房间

2、性能
速度从 110MB/s → 280MB/s，体验巨大提升
```

### 5.3 方案 3：半专业，NAS + 大文件传输

```
混合交换：
主路由(2.5G) → 万兆交换机（10G）
                  ↓ 10G
                 NAS
                  ↓ 10G/2.5G
                 电脑
                  ↓ 2.5G
                其他房间
```

## 六 总结

|        家用类型        |      选择      |
| :--------------------: | :------------: |
|        一般上网        |      千兆      |
|   有 NAS，想提升体验   | 2.5G（最推荐） |
| 大文件/剪辑/高性能 NAS |      10G       |
|     不知道选哪种？     | 2.5G 最稳最香  |
