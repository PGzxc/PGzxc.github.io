## 二 什么是交换机(Switch)

### 2.1 概念

```
交换机是一种局域网设备，用来把多台设备连接在一起，让它们在同一个网络中高速通信。
它是家庭和企业网络的“中心枢纽”。
```

### 2.2 交换机解决的问题

```
1.路由器 LAN 口不够？用交换机扩容
2.多设备之间需要高速传输？用交换机连接
3.NAS、电脑、电视盒子互传太慢？换千兆/2.5G/10G 交换机
```

## 三 交换机的核心作用

### 3.1 扩展网口(最基本作用)

```
路由器通常只有 1–4 个 LAN 口。
如果你还要连接：NAS\电脑\电视\AP（WiFi 接入点）\摄像头\游戏机，端口就不够用了。
交换机 = 多口网络插排，让你扩展成 8/16/24/48 个网口。
```

### 3.2 提升局域网速度(重要)

1、说明

```
多数设备互传都走 局域网（LAN）
```

2、交换机类型对比(交换机的速率决定你 LAN 的速度)

| 交换机类型  |  速度   |           适用场景            |
| :---------: | :-----: | :---------------------------: |
| 千兆交换机  |  1Gbps  | 家庭基础 LAN（NAS、电脑足够） |
| 2.5G 交换机 | 2.5Gbps |    NAS / WiFi 7 / 高速备份    |
| 10G 交换机  | 10Gbps  |   发烧级 NAS、工作室、影棚    |

3、例子(NAS拷贝 100GB 文件)

```
-千兆：约 10 分钟
-2.5G：约 4 分钟
-10G：约 1 分钟以内

差距非常明显。
```

### 3.3 精准转发数据(对比 Hub)

```
1、说明
交换机会学习设备的 MAC 地址表

2、当 A → B 发送数据时：
-Hub 会把数据广播给所有人
-交换机 只发给 B(目标 MAC)

3、特征
-更安全
-更快
-没冲突
```

### 3.4 VLAN 网络分区(管理型交换机特有)

```
1、说明
管理型交换机可把一个物理交换机分成多个虚拟网络：

2、示例
-VLAN 10：办公室
-VLAN 20：员工宿舍
-VLAN 30：访客网络

3、优点：
-安全隔离
-不同区域互不影响
-性能更高

家用一般不需要，但轻办公环境很常用。
```

### 3.5 PoE 供电(可选)

```
PoE：Power over Ethernet
一根网线同时传输 网络 + 电力
可直接给：摄像头\WiFi AP\门禁\电话机
供电 + 网络一线搞定，不需要插座。
```
