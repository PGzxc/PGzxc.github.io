## 二 交换机与路由器的区别

### 2.1 对比

|      对比项      |    交换机（Switch）    |         路由器（Router）          |
| :--------------: | :--------------------: | :-------------------------------: |
|     主要作用     | 扩展 LAN，连接内部设备 |        管控网络、NAT、上网        |
|     工作层级     |  二层（MAC） ➜ 局域网  |    三层（IP） ➜ 局域网 + 外网     |
|   是否分配 IP    |          不会          |            会（DHCP）             |
| 是否提供上网功能 |         不提供         |            提供（WAN）            |
|     端口用途     |    全部都是 LAN 口     |         WAN（上网）+ LAN          |
|   数据转发规则   | 按 MAC 地址表 精准转发 | 按 IP 路由 处理，支持 NAT、防火墙 |
|   可否隔离网络   |  VLAN（管理型交换机）  |      子网、DHCP、NAT、防火墙      |
|     典型速度     |    1G / 2.5G / 10G     |     多为 100M / 1G / Multi-G      |
|   是否有 WiFi    |          没有          |          有（无线功能）           |

### 2.2 一句话总结

```
路由器管“外网”
 交换机管“内网”
```

## 三 为什么一定要两个设备一起使用？

### 3.1 路由器和交换机功能不同

```
路由器 = 网络大脑（负责上网、IP、NAT）
交换机 = 网络身体（负责连接和高速传输）

你无法用交换机上网，也无法用路由器扩展到几十个网口。
```

### 3.2 网络结构

```
光猫 → 路由器（负责上网） → 交换机（扩容、提速） → 各设备
```

## 四 交换机的使用场景(Switch)

### 4.1 场景 1：扩展 LAN 口(最常见)

```
路由器只有 1–4 个 LAN 口，设备多了就不够了：
NAS/电视盒子/电脑/AP/摄像头/打印机
加交换机立刻扩容成 8/16/24/48 口。
```

### 4.2 场景 2：局域网高速传输(NAS 必备)

```
使用千兆/2.5G/10G 交换机可大幅提升传输速度。

例如 NAS 备份、传大文件：
-千兆：110MB/s
-2.5G：280MB/s
-10G：1000MB/s 以上
```

### 4.3 场景 3：PoE 供电(摄像头/AP)

```
PoE 交换机可用一根网线同时给设备供电和传输网络。

适用：
-摄像头（IPC）
-WiFi AP（如 Ubiquiti、H3C、TP-LINK）
-门禁
-对讲机
```

### 4.4 场景 4：VLAN 网络隔离(管理型交换机)

```
可把一个交换机分成多个虚拟网络：
-办公网络
-访客网络
-家庭 IoT 设备网络
-NAS 私有网络

提高安全性 + 性能。
```

## 五 路由器的使用场景(Router)

### 5.1 场景 1：让你可以上网(最核心)

```
路由器负责：
-将 WAN（外网）转换为 LAN（本地网络）
-NAT（多个设备共享一个公网 IP）
-DHCP（分配内网 IP）
-防火墙

交换机不具备这些功能。
```

### 5.2 场景 2：管理网络中的设备

```
路由器能控制：
-哪些设备能上网
-限速（QoS）
-端口转发（NAS 外网访问必用）
-DHCP 保留地址
-多子网设置（企业）
```

### 5.3 场景 3：WiFi 覆盖(带无线路由功能)

```
带 WiFi 功能的路由器可以：
-2.4G/5G/6G WiFi
-Mesh 分布式组网
-访客网络

交换机不提供 WiFi。
```

### 5.4 场景 4：VPN、DDNS、端口映射

```
如果你有 NAS，需要：
-外网访问 NAS
-远程家里电脑
-VPN 回家网络

这些都由路由器（或软路由）完成。
```
