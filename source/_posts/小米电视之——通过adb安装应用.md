## 一 准备工作

```
1.一台小米电视/小米盒子(已开机、连好 WiFi)
2.一台电脑(Windows/macOS 均可)，与电视在同一个 WiFi 下
3.要安装的.apk 文件(提前下载到电脑，老款电视+盒子v7a，新款v8a且兼容v7a)
4.电脑上装好 ADB 工具
```

<!--more-->

## 二 电视端(开启"开发者选项"与"网络 ADB 调试")

### 2.1 打开"开发者选项"

```
1.电视进入：设置 → 关于 → 产品型号 / 版本号
2.连续点击 产品型号 / 版本号 7 次左右，会提示 “已开启开发者选项”。
```

### 2.2 开启"网络 ADB 调试"

```
1.返回设置 → 开发者选项
2.找到并打开：
 -USB 调试
 -网络 ADB 调试（必须开，否则电脑连不上）
3.部分新版系统路径可能是：设置 → 账号与安全 → ADB 调试 → 开启
```

### 2.3 查看电视 IP 地址

```
1.设置 → 网络 → 已连接的 WiFi → 查看 IP（例如：192.168.8.97）
2.记下来，后面要用
```

## 三 电脑端操作(安装ADB并连接电视以及安装及卸载)

### 3.1 安装adb

1-Windows端

```
1.下载一个ADB 便携包
2.把下载的 platform-tools 解压到一个文件夹，比如 D:\adb
3.在该文件夹里，按住 Shift + 右键 → 在此处打开 PowerShell / 命令窗口
```

2-macOS/Linux

```
1.安装 Homebrew(如未装)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2.安装adb
brew install android-platform-tools

3.打开终端即可直接用 adb 命令
```

### 3.2 连接电视

```
1.在电脑命令行 / 终端输入
adb connect 你的电视IP:5555
示例:adb connect 192.168.8.97:5555

2.成功提示：connected to 192.168.1.105:5555
3.电视上会弹出 “允许调试吗？” → 勾选一律允许 → 确定
```

### 3.3 安装 APK

```
1.输入adb install 然后拖入apk文件并回车安装
2.成功提示：Success
3.安装完成后，在电视 “应用” 或 “我的应用” 里就能找到。
```

### 3.4 卸载APK

```
1.通过如下指令查看包名
adb shell pm list packages

2.执行如下指令，卸载APK
adb uninstall 包名
```

## 四 常见问题及解决

### 4.1 提示：cannot connect / 连接超时

```
-确认电脑和电视同一路由器
-电视端确认：网络 ADB 调试已开启
-防火墙 / 杀毒软件临时关闭再试
-重启电视、重启路由器后重试
```

### 4.2 提示：install failed / 解析包错误

```
-APK 损坏：重新下载
-APK 不兼容电视架构（电视一般是 arm64-v8a）
-空间不足：清理电视存储
```

### 4.3 安装成功但电视找不到应用

```
1.小米电视会隐藏部分第三方应用
2.解决：
安装一个桌面 / 应用启动器（如当贝桌面、电视应用管家）
或在电视 “应用商店” 里搜 “应用管理” 类工具打开
```
