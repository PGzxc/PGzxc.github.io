## 二 示例

### 2.1 布局文件(button.wxml)

```
<button type="{{isactive?'primary':'warn'}}" bindtap="buttonClick">{{isactive?'确认':'取消'}}</button>
```

### 2.2 逻辑文件

```
Page({
  data: {
    isactive: true
  },
  buttonClick() {
    this.setData({isactive: !this.data.isactive})
  },
})
```
