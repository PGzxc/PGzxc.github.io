## 二 开孔位置

### 2.1 开孔位置(推荐)

```
位置：楼梯间右侧墙上，靠上方（距天花板 10–20cm）处开孔
原因：湿热空气往上走，上部开孔效果最好。
```

### 2.2 开孔中心点高度

```
距离楼梯间上半段平台位置 上方 20–30cm
距离天花板 10–20cm: 便于风机安装，不会顶住梁
```

### 2.3 开孔直径 / 尺寸

```
根据选择设备不同：

1、选 1：排气扇（最佳）
孔径：160mm（常规）或 200mm(更强)
推荐：160mm 适合大多农村楼梯间
原因：风量足，墙体结构压力更小

2、选 2：百叶窗（无动力，自然风）
-尺寸建议：200×200mm 或 250×250mm 矩形孔
-百叶窗可遮雨、防虫，但效果 < 排气扇。
```

## 三 具体开孔步骤(农村自建房常用方法)

### 3.1 准备阶段

```
1、画线
-用水平仪在墙面标好中心点
-根据排气扇尺寸画圆（160mm）或正方形（200×200mm）

2、检查墙体
-确保不是承重梁
-墙体通常是红砖或水泥砖，可直接开孔
```

### 3.2 施工步骤(两种方法)

1、方法1：专业开孔机(优选)

```
1、说明：如果你能请附近工人来，一般 80–150 元搞定。

2、步骤：
-开孔机在画线中心定位
-冷水降温
-1次钻穿墙体(砖墙一般20–40秒)
-清理孔边
-套 PVC 管/风筒
-打发泡胶固定
```

2、方法2：家用工具自己开(可 DIY)

```
1、说明：
适用于砖墙（非钢筋混凝土）。

2、需要工具：
冲击钻 + 金刚石开孔器（150–180mm）

3、步骤：
-用小钻头沿着圆线钻 10–20 个浅孔
-换大孔开孔器一点点切削
-用锤子轻敲击碎
-清理碎砖
-套 PVC 管 / 安装排气扇
-用水泥砂浆修边
```

## 四 风机或百叶窗选择(明确推荐)

### 4.1 方案 A：排气扇(强力除潮，最推荐)

```
1、说明：适合你楼梯间这种 密闭+潮湿+返碱 的情况。

2、推荐参数：
-尺寸：150–200mm
-风量：250–350 m³/h
-功率：18–28W
-静音：35–45 dB

3、推荐类型：
-厨房/卫生间 专用壁式排气扇
-自动闭合百叶（防雨/防虫）
```

### 4.2 方案 B：百叶窗(自然通风)

```
优点：便宜不耗电
缺点：湿气重时效果弱

推荐尺寸：200×200 或 250×250 mm
```
