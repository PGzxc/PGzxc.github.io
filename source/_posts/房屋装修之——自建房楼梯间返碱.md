## 二 楼梯间返碱原因

### 2.1 原因分析

```
楼梯间是封闭空间 + 温差大 + 下方厨房/客厅湿气上升 → 墙体长期“冷凝水” → 腻子起皮返碱
```

### 2.2 结构如下

```
-楼梯间 没有通风
-外墙是裸砖/混凝土 → 冬天外墙冷
-楼下是厨房/客厅 → 热气、湿气上升
-楼梯间像一个“烟囱”一样聚湿
-热湿空气碰到冷墙 → 墙面大量结露（看不见但湿）
-久了 → 腻子层粉化、起皮、返碱

这与漏水无关，与外墙防水也无关，是典型“内墙冷凝潮湿返碱”。
```

## 三 楼梯间返碱治理方案

### 3.1 方案 A(最推荐)-内墙防潮 + 封闭底漆 + 重新批腻子刷漆(根治版本)

```
1、把所有松动、爆点、起皮腻子全部铲干净
-铲到灰色放心位置（砂浆层）。

2、用白醋或除霉剂擦洗返碱霉点(必做)
-返碱要中和掉，不然涂料还是会被顶开。

3、墙面喷刷“防潮封闭底漆”(双遍)

这是核心！！！

推荐：
-立邦抗碱底漆（NP10）
-多乐士专业抗碱底漆
-雨虹墙面封闭底漆

刷2遍，封闭碱性 + 减少潮气进入腻子层。

4、砂浆基层松的地方要刷墙固/界面剂
-保证后续腻子能牢牢粘住。

5、批腻子 2~3 遍 + 打磨找平

6、刷乳胶漆 2 遍
推荐选防潮型的，如：
-多乐士净味防潮
-立邦净味防潮
```

### 3.2  方案 B(更彻底)—增加楼梯间“通风”

```
1、原因：
因为你的楼梯间是 封死的空间，湿气无法排走，所以长期潮湿。

2、如何通风
2-1、在楼梯间顶部装一个 小型静音排气扇
几十块钱即可，让空气流动起来。

2-2、或者在楼梯间顶部开一个 百叶窗孔
让湿气自然排出。

通风一做，你这个位置以后都不会再发生返碱
```
