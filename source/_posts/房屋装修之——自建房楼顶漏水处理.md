
## 二 判断你是哪种漏水

先搞清楚是哪种情况，不然材料花了也白做：

### 2.1 表面裂缝漏水(最常见)

```
1、现象
-楼顶面层出现龟裂、细小裂纹
-下雨后渗水、返潮

2、处理
用：JS 防水 + 裂缝处理即可
```

### 2.2 楼板结构性裂缝

```
1、现象
-裂缝较深，有“开口”
-下雨大时漏得厉害

2、处理
先补裂缝，再整体做防水
```

### 2.3 管根、落水口漏水

```
1、现象
-落水口周围漏
-管子周围空隙大

2、处理
重点加强这些节点
```

## 三 农村自建房最稳妥方案(推荐)

这是工人师傅最常用、返工率最低的做法：

### 3.1 步骤 1：清理楼面

```
清掉浮灰、青苔、松动水泥
必须保证楼面干净
```

### 3.2 步骤 2：处理所有裂缝

```
1、方案 A(最佳)：环氧树脂灌缝
-适用于结构裂缝、深裂缝
-直接灌注裂缝，比水泥更牢
-封死裂缝根部

2、方案 B(便宜)：聚氨酯或堵漏王填补
-适用于浅裂缝
-对裂缝开 V 字槽
-用聚氨酯或堵漏王填充
```

### 3.3 步骤 3：整体做防水(关键)

```
农村顶通常用 JS 防水 或 聚氨酯防水，推荐效果排序：

1、最耐用：单组份聚氨酯防水(寿命8~12年)

特点：
-拉伸好，堵裂效果比JS 好
-雨淋不怕
-贵一点，但很稳

2、经济型：JS 防水(寿命3~6 年)

特点：
-价格便宜
-刷 2~3 道
-必须加网布加固（玻纤布/无纺布）
```

### 3.4 步骤 4：做保护层(非常重要)

```
只刷防水 ≠ 完工
暴晒 1 年就会老化 → 必须加保护层

两种保护层：

1）水泥砂浆保护层(最常见)
-厚 2~3cm
-最便宜、最耐晒
-防水层才不会被太阳烤坏

2）瓷砖保护层(更耐用，但贵)
-贴地砖、美观
-能保 10 年+
-不做保护层 = 白做！农村楼顶防水返工率 90% 是因为没做保护层。
```

## 四 特殊部位的处理(这是漏水最多的地方)

### 4.1 落水口

```
必须用 聚氨酯加强
做成一个“喇叭口”，避免积水
```

### 4.2 管根

```
用密封胶 + 聚氨酯包裹
再贴网布
```

### 4.3 阴阳角(墙角)

```
用水泥打成圆角
防水要多刷 2 遍
```

## 五 最推荐的材料(农村常买得到)

### 5.1 防水

```
东方雨虹 单组份聚氨酯（灰桶）
德高 JS 防水
西卡 Sika 防水
```

### 5.2 裂缝填补

```
灌缝胶(环氧)
德高堵漏王
聚氨酯密封胶
```
