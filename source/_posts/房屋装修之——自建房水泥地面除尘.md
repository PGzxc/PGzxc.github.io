## 一 概述

```
本文介绍：
-最清晰、最实用、完全可自己施工 的《混凝土地坪固化剂施工指南》。
-按这个流程走，农村院子、水泥地、车库都能一次成功、无灰不起砂。
```

<!--more-->

## 二 准备阶段—工具准备(简单)

```
-喷壶/园艺喷洒壶(推荐)
-毛滚筒(可选)
-硬毛刷/地板刷(可选)
-胶刮(可选)
-水桶 + 清水
-扫把
```

## 三 施工流程(最重要部分)

### 3.1 清洁地面(关键)

```
清理所有灰尘、垃圾：
-扫干净
-若地面很脏，可用水冲洗后干透
-若有油污，必须用洗洁精+热水刷掉（重要）

越干净，效果越好。
```

### 3.2 喷洒固化剂(核心步骤)

```
1、方法：
-使用喷壶均匀湿喷
-像喷水一样，把地面喷“湿润但不积水”

2、喷涂标准：
-涂布量：0.1–0.3 L/㎡（一平方米100–300ml）
-效果：地面颜色整体变深，看起来“发湿”

3、注意：
地面必须始终保持 15–30 分钟的“湿润状态”，不要让它很快干掉。
```

### 3.3 静置渗透(15–40 分钟)

```
固化剂会自动渗入混凝土。

观察地面：
-若10 分钟内就吸干 → 说明表层粉化严重 → 要补喷
-若一直不干、一直湿湿的 → 用刮板把多余的刮开即可
```

### 3.4 刷洗/搓磨(可选，但效果更好)

```
若地面为老旧粉化严重，可：
-用硬毛刷在地面“来回搓几下”
-促进固化剂渗透更深

新地面可以省略。
```

### 3.5 再次喷洒(老地面必做)

```
如果地面非常粉化：建议喷 2 次。

做法：
-第1遍干后再喷第2遍
-第2遍只需轻喷即可
```

### 3.6  冲洗 & 擦干(很多人不知道要做)

```
1、说明
-固化剂反应后会在表面留下结晶层
-如果不洗掉，会有点白雾感。

2、建议：
-施工后 1–2 小时，用清水轻轻淋（或湿拖）一次
-均匀清洁表层

3、效果：
-表面更亮
-不滑
-不留白印
```

### 3.7 等待干燥(4–12 小时)

```
4 小时可走人
12–24 小时达到硬化初期
7 天达到最佳强度
```

### 3.8 可选：打蜡或抛光(让地面更亮)

```
1、不是必须，但能提升：
-光泽
-易清洁性
-使用寿命

2、工具：普通家用打磨机或抛光机
```

## 四 注意事项(必须看)

```
1. 不要一次喷太多
-避免“积液”，保持湿润即可，不要变成一滩水。

2. 不要在大太阳下施工（重要）
-暴晒会导致固化剂没来得及渗透就被晒干。
-最佳时间：上午 8–10 点、下午 4–6 点

3. 不要在地面有油污时施工
-油会阻挡渗透。

4.不要下雨时施工
-雨水会影响化学反应。
```

## 五 自施工用量计算

### 5.1 买多少固化剂？

```
公式：材料量（升） = 面积 × 0.15–0.3
```

### 5.2 举例

```
50㎡：约 7–10 L
100㎡：约 15–20 L

老地面按 0.3 计算
新地面按 0.15–0.2 计算
```
