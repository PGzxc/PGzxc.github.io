## 一 单词一

### 1.1 列表

|   假名   | 日本汉字 |    中文意思    |     示例     |
| :------: | :------: | :------------: | :----------: |
|   どう   |          | [副]怎样，如何 |  どうですか  |
|  いかが  |          |  [副]如何(礼)  | いかがですか |
| いろいろ |          |  [副]各种各样  |              |
|   でも   |          | [连]可是，不过 |              |
|  そして  |          | [连]而且，可是 |              |

<!--more-->

### 1.2 生词或词语

* どうですか：情况怎么样呀？
* どう食べますか：这个怎么吃呀？
* いろいろだ：いろいろ原型(二类动词)
* この料理は美味しいです。そして　安いです——这道菜味道很好，而且价格便宜
* 料理(りょうり)：料理
* 美味しい(おいしい)：美味
* 安い(やすい)：便宜
