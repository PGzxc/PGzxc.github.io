### 2.2 动词说明

| 动词类别 |  如何分类   | 如何变换 |     示例     |   肯定/否定/疑问   |
| :------: | :---------: | :------: | :----------: | :----------------: |
| 一类动词 |   词尾3段   | 3段变2段 | 行くー＞行き | ます/ません/ますか |
| 二类动词 | 2段、4段+る |          |              |                    |
| 三类动词 |             |          |              |                    |

注：  

- 行く：いく
- チーズ：奶酪(拍照时说)
- 帰る(特殊一类动词)：かえるー＞かえり
- 来る：去掉る，加ます，く变き

## 五 短语

* お疲(つか)れさまでした：够累的
* お先に　失礼します——我先走了、再见
* あるいて(歩いて)——步行
* たいへんですね(大変ですね)——真不容易，够受了，不得了

注：  

* 先：さき
* 失礼：しつれい
* 大変：たいへん——够受的，不得了
