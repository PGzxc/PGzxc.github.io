## 二 常见系统格式

购买移动硬盘(例如 Western Digital 4TB)后，很多人都会遇到一个问题：应该使用什么文件系统格式？

### 2.1 不同文件系统在以下方面差异很大

```
1.操作系统兼容性
2.单个文件大小限制
3.是否支持 NAS / 路由器
4.是否支持权限管理
5.适合的使用场景
```

### 2.2 常见文件系统包括

```
NTFS
exFAT
FAT32
EXT4
APFS / HFS+
```

## 三 常见移动硬盘文件系统对比

### 3.1 对比表格

| 文件系统 | 单文件限制 | Windows | macOS | Linux | 路由器/NAS |   推荐用途   |
| :------: | :--------: | :-----: | :---: | :---: | :--------: | :----------: |
|  FAT32   |    4GB     |    ✔    |   ✔   |   ✔   |     ✔      | 老设备、车机 |
|  exFAT   |   无限制   |    ✔    |   ✔   |   ✔   |     ✔      |    跨平台    |
|   NTFS   |   无限制   |    ✔    | 只读  |   ✔   |  部分支持  |   Windows    |
|   EXT4   |   无限制   |    ✖    |   ✖   |   ✔   |     ✔      | NAS / Linux  |
|   APFS   |   无限制   |    ✖    |   ✔   |   ✖   |     ✖      |     Mac      |

### 3.2 说明

```
如果是 普通移动硬盘用户，最常见的是：
-NTFS
-exFAT
```

## 四 几种格式说明

### 4.1 FAT32 文件系统

```
一、特点
FAT32 是最早期的通用文件系统之一。

1.1-优点：
兼容性极好
几乎所有设备都支持

1.2-缺点：
单个文件最大 4GB
不适合高清视频

二、适用设备

2.1-适合：车载播放器/老电视/老路由器/游戏机/U盘
2.2-例如：老款电视/老路由器 USB 共享

三、使用场景

3.1-推荐用于：U盘/小文件传输/老设备兼容
3.2-不推荐：存储电影/NAS存储
```

### 4.2 exFAT 文件系统(最推荐)

```
-安卓：支持一、特点
exFAT 是微软为移动存储设计的格式。
1.1-优点：
-无 4GB 限制
-Windows / Mac 全支持
-适合移动硬盘

1.2-缺点：
-权限功能较弱
-部分老设备不支持

二、系统兼容性
-Windows：支持
-macOS：支持
-Linux：支持
-安卓：支持

三、路由器支持
-很多路由器支持 exFAT，例如：ASUS 路由器、TP-Link 路由器
-常见功能：SMB共享/FTP/DLNA
-例如：移动硬盘 → 路由器USB → 局域网访问

四、推荐用途

最推荐：
-移动硬盘 
-跨 Windows / Mac 使用
-电视播放电影
```

### 4.3 NTFS 文件系统

```
一、特点
NTFS 是 Windows 默认文件系统。

1.1-优点：支持权限/支持压缩/稳定性高
1.2-缺点：Mac 默认 只读

二、系统兼容性
-Windows：支持
-macOS：只读
-Linux：支持

说明：Mac 如果要写入，需要安装工具；例如：Paragon NTFS、Tuxera NTFS

三、路由器支持
-很多路由器支持 NTFS：例如：ASUS RT‑AC66U B1
-但可能存在问题：写入速度慢/兼容性问题

四、推荐用途

-Windows电脑
-NAS数据盘
-大文件存储
```

### 4.4 EXT4 文件系统(NAS常用)

```
一、特点
EXT4 是 Linux 默认文件系统。

1.1-优点：稳定/高性能/NAS常用
1.2-缺点：Windows 不支持/Mac 不支持

二、常见设备

很多 NAS 使用 EXT4，例如：Synology NAS/QNAP NAS/WD My Cloud

三、推荐用途
3.1-适合：NAS/Linux服务器/Docker数据盘
3.2-不适合：移动硬盘
```

### 4.5 APFS / HFS+(Mac 专用)

```
1.苹果系统专用格式：
-Apple File System
-HFS+

2.优点：Mac性能最佳
3.缺点：Windows 不支持
4.适合：Mac外置硬盘
```

## 五 不同场景推荐

### 5.1 移动硬盘(最推荐)

```
1.推荐：exFAT

2.原因：
-Windows / Mac 通用
-无文件大小限制
```

### 5.2 Windows电脑专用

```
1.推荐：NTFS
2.优点：稳定/权限支持
```

### 5.3 路由器共享硬盘

```
1.推荐：exFAT 或 NTFS
2.如果路由器较老：FAT32
```

### 5.4 NAS硬盘

```
推荐：EXT4
```
