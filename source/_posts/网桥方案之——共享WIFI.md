## 一 要求说明

```
两家相距：500米
需求：两家都能上网、稳定
环境：户外，有风吹日晒雨淋
条件：其中一家已部署网络
目的：另外一家使用WiFi网络
```

<!--more-->

## 二 方案选择(无线网桥)

### 2.1、场景分析

```
-两家距离：约 500 米
-房屋高度：2~3 层
-地形：可能有树、建筑物挡住视线
-中间没电：不能用有线中继
```

### 2.2 、首选方案:5GHz 无线网桥(穿透+对准安装)

|     设备     |          推荐型号          |                   说明                   |
| :----------: | :------------------------: | :--------------------------------------: |
| 无线网桥一对 | TP-Link TL-CPE605、腾达 O6 | 300Mbps，传输距离可达 2~5 公里，PoE 供电 |
|  PoE 适配器  |          随机附带          |          用于供电(通过网线供电)          |
| 室外防水网线 |     Cat6 20 米 × 2 根      |              连接路由与网桥              |
|     支架     |      铁杆 / 墙壁支架       |              固定并调整方向              |

### 2.3 、安装方式图解

```
（家A）                                      （家B）
┌───────────────┐                    ┌───────────────┐
│ 光猫/路由器     │──网线──┐         ┌──网线──┤ 家B路由 │
└───────────────┘        │         │          └───────────────┘
                         │         │
                   [PoE供电器]    [PoE供电器]
                         │         │
                  ┌──────────┐    ┌──────────┐
                  │ 网桥A端  │ )))) 信号 ((((( │ 网桥B端 │
                  └──────────┘    └──────────┘
                  ↑ 2~3楼阳台/屋顶  ↑ 2~3楼阳台/屋顶


家B 的路由设置成「自动获取 IP」或「AP 模式」，即可上网
```

### 2.4 、预算

```
约 300~600 元 全搞定，速度和延迟几乎等同有线连接
```

### 2.5 安装要点

```
1、安装高度
-两边尽量安装在屋顶或二层阳台栏杆上
-确保两设备之间大致能“看到对方方向”（无遮挡最佳）

2、对准方向
-开机后进入网桥管理界面
-打开「信号强度指示」或手机 APP（TPLINK 也有工具），慢慢调整方向，直到信号强度 ≥ -60dBm

3-网段设置
-主网桥（A 端）设置为 AP 模式
-从网桥（B 端）设置为 Client 模式
-家B的路由器设置为「DHCP 自动获取IP」或「AP模式」

4-防水/避雷
-接口用防水胶布缠好
-如条件允许，设备接地
```

### 2.6 网络性能预估

|   项目   |                 指标                  |
| :------: | :-----------------------------------: |
| 实测速率 |   200~300 Mbps(取决于设备与对准度)    |
|   延迟   |                2~5 ms                 |
|  稳定性  |      下雨天信号略降，但基本可用       |
| 适合场景 | 家庭宽带共享、NAS访问、IPTV、监控传输 |

### 2.8 方案增强建议

```
如果地形较复杂（例如中间有树挡信号）：
-可以升高安装点（用 3~4 米铁杆或水管固定）
-或使用 Ubiquiti NanoStation Loco M5 / LiteBeam M5（传输能力更强，约¥600~¥700）
-若家B信号不稳定，可加一个 AP（无线扩展器） 覆盖全屋
```
