## 一 场景及要求

```
场景是：A屋有宽带+WiFi，B屋在后方约500米，无电中继点，要求信号稳定、美观、不破坏建筑结构。
目标：让B屋能像在A屋一样稳定上网（看视频、上网、IPTV均不卡）。
```

<!--more-->

## 二 推荐方案：无线网桥 + 隐藏安装 + 室内接收路由

### 2.1 核心思路

```
用一对“点对点无线网桥”（俗称无线光纤）在A、B两屋之间建立稳定无线通道，相当于“看不见的网线”。
```

### 2.2 简图

```
A屋 ⇄（无线信号500米）⇄ B屋→ B屋再接普通路由器/交换机使用。
```

## 四 安装布局(不破坏结构)

```
1、A屋（有宽带）
-将发射端网桥固定在窗边/屋檐下（对着B屋方向）。
-用PoE供电器连接A屋路由器LAN口。
-不需打孔，可使用窗外支架或夹具固定。

2、B屋(无电中继)
若B屋有电插座则更好；如确实没有电，可选以下两种方式：
-太阳能供电：买个小型太阳能+电池套件（12V/2A）；
-PoE供电延长线：若中间路程不超80米，可由A屋拉PoE电缆为B屋设备供电。

然后在B屋端：
-安装接收端网桥（对准A屋方向）；
-接一台普通路由器（WAN口接收端网线）；
-B屋内部WiFi覆盖即可。
```

## 五 信号优化与隐蔽技巧

```
-安装高度建议≥2米，避开大树、铁皮墙；
-设备体积仅约手掌大小，可藏于雨棚或空调外机旁；
-使用白色外壳款，与墙体融合不明显；
-建议使用 5GHz 设备（如Tenda O3）避免2.4G干扰；
-配置好静态IP，长期稳定运行。
```
