## 二 FFmpeg 变音方案

```
1、下载 FFmpeg：官网。

2、提取原视频音频：
ffmpeg -i input.mp4 -vn -acodec pcm_s16le -ar 44100 -ac 1 audio.wav。

3、变音示例命令(调整 pitch 为变调，atempo 为变速)
3.1、变高音(+2 半音)：
 ffmpeg -i audio.wav -filter:a "asetrate=44100*1.125,atempo=0.888" pitched_audio.wav。
3.2、变低音(-2 半音)：
ffmpeg -i audio.wav -filter:a "asetrate=44100*0.888,atempo=1.125" lowered_audio.wav。
3.3、变速(0.8 倍速)：
ffmpeg -i audio.wav -filter:a "atempo=0.8" slowed_audio.wav。

4、重新合成视频：
ffmpeg -i input.mp4 -i modified_audio.wav -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4。

5、这会保持视频不变，只改音频。
```

## 三 安卓变音方案

### 3.1 软件

```
Narrator Voice/Voice Changer
```

### 3.2 操作

```
-用 SmartSub 导出字幕 SRT
-视频抽音轨(可用“音频提取器 Audio Extractor”)
-用 App 变声(男声→女声/少年音/机器人等)
-再用“CapCut/剪映”把变声音轨 + 字幕 + 视频合成
```

## 四 IOS变音方案

### 4.1 方案 A：语音变声器 Voice Changer Plus

```
-提取音频(在 iPhone 可用 iOS“文件”App 或其他 App)
-导入 Voice Changer Plus
-选择变声类型
-导出变声后的音频
-用 CapCut/剪映 把音频 + 字幕合成
```

### 4.2 方案 B：CapCut(剪映)直接变音

```
1、CapCut 内置变声功能，
包括：少年、萝莉、大叔、电子/机器人、厚重声音、耳语

2、操作：
-打开 CapCut
-导入视频
-声音 → 音效 → 变声
-导出

3、字幕 SRT 可直接导入 CapCut！
```

## 五 Windows/Mac变音方案

### 5.1 方案 A：Adobe Audition 变声

```
-安装 Adobe Audition
-导入音频
-效果 → 音高移调 → 改变音色
-导出音频
-用 PR/AE/CapCut 把音频 + 字幕合成
```

### 5.2 方案 B：ElevenLabs/MetaVoice(AI 配音)

```
1、可实现
完全 AI 重录 + 拟真人声
年龄、性别都可调
可克隆你声音再调整

2、推荐：
-https://elevenlabs.io
-https://metavoice.ai

3、流程：
-SmartSub 导出字幕（SRT）
-用 SRT 内容 → AI 配音生成新音频
-合成视频

效果比 App 变声自然得多
```

## 六 完整实战流程示例(最简单)

```
假设你用手机：

Step 1：SmartSub 生成字幕
-导出 SRT

Step 2：CapCut 导入视频
→ 声音 → 变声（选择大叔/少女/可爱等）

Step 3：导入字幕
→ 文本 → 导入 SRT

Step 4：导出成品视频

整套流程最快只需 3 分钟。
```
