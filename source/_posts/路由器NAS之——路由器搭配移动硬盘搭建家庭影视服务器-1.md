## 一 概述

```
本文介绍：
 1.不用买 NAS
 2.移动硬盘 + 路由器搭建家庭影视服务器教程
```

<!--more-->

## 二 场景+原理+设备

### 2.1 需求

```
1.需求：很多家庭用户都有一个需求：
-移动硬盘里存了很多电影、电视剧
-希望手机、平板、电视都能观看
-又不想花几千元买 NAS

2.如何实现
其实只需要：
-移动硬盘 + 带 USB 的路由器
-就可以搭建一个 简易 NAS（家庭影视服务器）。

3.成本：基本为 0 元。
```


### 2.2 实现原理

```
一、连接结构

移动硬盘
    │
USB接口
    │
路由器（文件共享）
    │
WiFi / 网线
    │
手机 / 平板 / 电视 / 电脑

二、路由器开启下功能
1.Samba（SMB）共享
2.DLNA媒体服务器

这样局域网所有设备都可以访问硬盘内容。
```


### 2.3 设备要求

```
一、路由器必须支持 USB
1.例如：华硕 AC66U B1/华硕 AX86U/小米 AX6000/TP-Link XDR 系列
2.只要有 USB接口 + 文件共享功能 就可以。

二、移动硬盘
1.支持格式：NTFS/exFAT/FAT32
2.推荐：NTFS (因为支持大文件(电影通常 >4GB)。
```


## 三 搭建步骤(例：华硕路由器 )

### 3.1 第1步 插入移动硬盘

```
将移动硬盘插入路由器 USB 接口。
等待约 10秒。
```

### 3.2 第2步 登录路由器

```
1.打开浏览器输入：http://192.168.50.1
2.输入管理员账号密码登录。
```

### 3.3 第3步 打开 USB 应用

```
1.进入：USB 应用

2.可以看到：
-AiDisk
-服务器中心
-媒体服务器
```

### 3.4 第4步 开启文件共享(Samba)

```
1.进入：USB应用
→ 服务器中心
→ 网络共享（Samba）

2.开启：启用 Samba
这样电脑、手机就能访问文件。
```

### 3.5 第5步 开启 DLNA 媒体服务器(推荐)

```
如果是 电视播放电影，建议开启。

1.进入：
USB应用
→ 媒体服务器
→ DLNA

2.开启：
Enable Media Server
这样电视会自动发现媒体库。
```

## 五 播放效果

### 5.1 说明

```
1.取决于路由器 USB 速度。
2.一般 USB2.0：20MB/s ~ 30MB/s
```

### 5.2 播放效果

| 视频类型 |   播放情况   |
| :------: | :----------: |
|   720P   |     流畅     |
|  1080P   |     流畅     |
|    4K    | 部分可能卡顿 |

备注：如果是千兆路由器 + USB3.0，播放 4K 基本没问题

## 六 优缺点及适合使用场景

### 6.1 优缺点

```
一-优点
1. 成本低
2. 搭建简单
3. 不需要额外设备
4. 手机电视都能访问

二-缺点
1. 速度不如 NAS
2. 无 RAID 数据保护
3. 多人同时播放可能卡
4. 功能较少
```

### 6.2 适合使用场景

```
1.适合：
-家庭电影库
-存电视剧
-手机/电视播放
-个人文件共享

2.不适合：
-企业存储
-多人同时访问
-重要数据存储
```
