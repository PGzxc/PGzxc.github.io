## 二 三方库推荐(HarmonyOS 社区)

|            库名            |         功能         | 地址 |
| :------------------------: | :------------------: | :--: |
| @ohos.window（系统库） | 官方窗口与安全区 API | 系统自带 |
| ohos_safearea | 封装 SafeArea 与状态栏管理 | https://gitee.com/openharmony/applications_app_samples |
| ArkUIPlus | UI 增强库（含 SafeArea 与沉浸式控制） | https://github.com/ohos-dev/ArkUIPlus |
| hw-foundation/window-utils | 常用 Window 辅助封装 | 鸿蒙社区库 |

## 三 Android/iOS/鸿蒙 安全区对照表

|   平台    |    安全区域 API    |                 获取方式                  |          沉浸式模式           |
| :-------: | :----------------: | :---------------------------------------: | :---------------------------: |
|  Android  | WindowInsetsCompat |        View.getRootWindowInsets()         | WindowInsetsController.hide() |
|    iOS    |   SafeAreaInsets   |            view.safeAreaInsets            |      .ignoresSafeArea()       |
| HarmonyOS |     AvoidArea      | window.getWindowAvoidArea() / .safeArea() | setWindowSystemBarEnable([])  |

## 四 总结

|          内容           |             推荐方案             |
| :---------------------: | :------------------------------: |
| 自动避开状态栏 / 导航栏 |  .safeArea(SafeAreaType.SYSTEM)  |
|     获取安全区高度      |   window.getWindowAvoidArea()    |
| 动态监听折叠 / 键盘变化 |    win.on('avoidAreaChange')     |
|         沉浸式          |   setWindowSystemBarEnable([])   |
|       兼容多设备        | 动态获取 + SafeAreaType 自动适配 |
